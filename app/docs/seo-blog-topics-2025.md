## SEO Strategy Overview

### Content Model: Hub & Spoke
Each cluster has one **hub article** (comprehensive guide, 1,800-2,000 words) and multiple **spoke articles** (focused topics, 800-1,500 words). This establishes topical authority with Google.

### Positioning Types
| Type | Description | % of Articles |
|------|-------------|---------------|
| **Complement** | TutorLingua works WITH Preply/iTalki for student discovery | 40% |
| **Replacement** | TutorLingua replaces scattered tools (Calendly, Beacons, spreadsheets) | 32% |
| **Authority** | Pure educational value to build trust (no hard sell) | 28% |

### SEO Best Practices Applied
- Long-tail keywords (3-5 words) for lower competition
- Featured snippet optimization (Q&A format, clear headings)
- E-E-A-T signals (Experience, Expertise, Authoritativeness, Trustworthiness)
- Bilingual SEO with localized keywords (not direct translations)
- Internal linking to TutorLingua features

### Competitors Referenced
- Preply (primary)
- iTalki
- Verbling
