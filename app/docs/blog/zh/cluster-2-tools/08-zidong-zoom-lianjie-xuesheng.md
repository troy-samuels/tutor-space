# 如何自动向学生发送Zoom链接：完整指南

"老师，Zoom链接在哪里？"

厌倦了收到这条消息吗？课程开始前5分钟学生找不到链接手忙脚乱...

这个问题完全可以预防。设置自动化后，预约确认的那一刻Zoom链接就会自动创建并发送给学生。

本指南按最流行的工具分步介绍如何设置自动Zoom链接。

## 为什么自动Zoom链接很重要

### 手动链接的问题

手动发送Zoom链接时：

1. **浪费时间**：每次预约2-3分钟 × 每周20个预约 = 每周1小时
2. **会忘记**：忙的时候会忘记发链接
3. **混乱**：学生尝试使用旧链接
4. **延迟**：课程开始时间花在找链接上

### 自动化的好处

- **0分钟投入**：设置一次永远有效
- **一致性**：每次预约相同体验
- **专业性**：立即发送确认邮件
- **减少缺席**：链接始终包含，消除借口

## 方法1：Calendly + Zoom

最简单的设置。5分钟内完成。

### 前提条件
- Calendly账户（免费版可用）
- Zoom账户（免费版可用）

### 设置步骤

**第1步：在Calendly中连接Zoom**
1. 登录Calendly
2. 点击Account → Integrations
3. 找到"Zoom"点击"Connect"
4. 登录Zoom授权

**第2步：将Zoom应用到活动类型**
1. 转到Event Types
2. 点击要编辑的活动
3. 在"Location"部分选择"Zoom"
4. 保存

**第3步：测试**
1. 转到您的预约页面
2. 进行测试预约
3. 确认确认邮件中有Zoom链接

### 结果
- 学生预约时 → 自动创建唯一Zoom会议
- 确认邮件包含Zoom链接
- 提醒邮件也包含Zoom链接
- 自动添加到日历事件

## 方法2：Google Calendar + Zoom

如果已经使用Google Calendar可以直接集成。

### 前提条件
- Google Workspace账户（或个人Google账户）
- Zoom账户

### 设置步骤

**第1步：向Google Calendar添加Zoom**
1. 打开Google Calendar
2. 点击设置⚙️ → "Add-ons"
3. 搜索"Zoom for Google Workspace"并安装
4. 用Zoom账户登录

**第2步：向事件添加Zoom**
1. 创建新事件
2. 点击"Add Google Meet video conferencing"旁边的下拉
3. 选择"Zoom Meeting"
4. 保存

### 限制
- 每个事件需要手动选择Zoom
- 与预约系统分开（不如Calendly自动化）
- 需要手动向学生发送日历邀请

## 方法3：Acuity Scheduling + Zoom

Acuity比Calendly提供更多选项。

### 设置步骤

**第1步：连接Zoom**
1. 登录Acuity
2. Integrations菜单
3. 找到"Zoom"并连接

**第2步：应用到预约类型**
1. 转到Appointment Types
2. 编辑每种类型
3. 在"Location"中选择Zoom
4. 保存

### 额外选项
- 只对特定预约类型应用Zoom
- 设置备选视频工具（Google Meet、Microsoft Teams）
- 自定义Zoom设置（等候室、密码等）

## 方法4：TutorLingua + Zoom

TutorLingua是为导师设计的，设置最简单。

### 设置步骤

**第1步：连接视频设置**
1. TutorLingua仪表板 → Settings
2. 点击"Video"标签
3. 点击"Connect Zoom"
4. 登录Zoom授权

**第2步：完成！**
- 所有新预约自动创建Zoom链接
- 包含在确认邮件中
- 包含在提醒中
- 学生门户可直接访问

### TutorLingua的额外功能
- 所有课程录制（Studio计划）
- 课程笔记与Zoom链接连接
- 在学生课程历史中查看链接

## 替代方案：固定Personal Meeting ID

不是每次创建新会议，也可以使用固定链接。

### 设置方法
1. 登录Zoom → Profile
2. 确认"Personal Meeting ID"
3. 在所有预约确认中使用此链接

### 优点
- 无需设置
- 始终相同链接
- 学生容易记住

### 缺点
- 安全性弱（任何人都可以进入）
- 预约时间重叠时冲突
- 会议记录混在一起

### 建议
- **1:1常客学生**：Personal Meeting ID可以
- **新学生/团体**：推荐自动生成的唯一链接

## Zoom替代方案：其他视频工具

除了Zoom还有其他选择：

### Google Meet
- 优点：只需Google账户免费，无时间限制
- 缺点：录制需付费（Google Workspace）
- Calendly集成：支持
- Acuity集成：支持
- TutorLingua：支持

### Microsoft Teams
- 优点：Microsoft 365用户熟悉
- 缺点：邀请外部用户可能复杂
- Calendly集成：支持
- Acuity集成：支持
- TutorLingua：支持

### Whereby
- 优点：无需安装应用，直接在浏览器中
- 缺点：免费计划有限
- 外部集成：有限

### 推荐
- **大多数导师**：Zoom（最通用，可靠性高）
- **Google用户**：Google Meet
- **企业学生多**：Microsoft Teams

## 故障排除

### "Zoom连接不上"
1. 确认已登录Zoom账户
2. 清除浏览器缓存重试
3. 尝试其他浏览器
4. 在Zoom应用中登出后重新登录

### "链接没出现在邮件中"
1. 确认活动类型的Location设置为"Zoom"
2. 确认邮件模板包含{location}或{zoom_link}变量
3. 重新进行测试预约

### "学生进不来"
1. 确认链接正确复制
2. 检查等候室设置（学生可能在等待中）
3. 确认是否需要密码
4. 建议更新Zoom应用
