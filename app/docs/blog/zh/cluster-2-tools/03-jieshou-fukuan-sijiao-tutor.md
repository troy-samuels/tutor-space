# 私人语言导师如何接受付款：完整指南

"您要怎么付款？"如果您对这个问题的回答是"嗯...您可以PayPal给我..."，那么您正在错失商机。

专业的支付处理不仅仅是收钱：
- **建立信任**：专业的支付选项提高信任度
- **改善收款**：简单的支付 = 更快的付款
- **减少缺席**：预付款提高出席率
- **节省时间**：无需追踪发票、发送提醒

本指南比较了私人语言导师的所有支付选项，帮助您选择最适合您情况的方式。

## 支付选项概述

### 选项1：PayPal

**最熟悉的国际选项**

PayPal是全球最受认可的在线支付方式。学生很可能已经有账户。

**费用：**
- 国内交易：3.49% + 固定费用（¥2.5或$0.49）
- 国际交易：4.49% + 固定费用
- 货币转换：+2.5%

**优点：**
- 全球认可
- 学生熟悉
- 提供买家保护
- 包含发票功能

**缺点：**
- 费用较高（尤其是国际）
- 账户冻结风险
- 汇率不利
- 需要商业账户

**示例计算（¥300课程）：**
- 国内：¥300 - ¥10.47 - ¥2.5 = ¥287 收到
- 国际：¥300 - ¥13.47 - ¥2.5 = ¥284 收到

**适合人群：**
- 有很多国际学生的导师
- 需要快速设置的情况
- 已经在使用PayPal的人

### 选项2：Stripe

**专业人士的标准**

Stripe是最流行的在线支付处理服务，许多平台使用Stripe。TutorLingua也集成了Stripe。

**费用：**
- 标准：2.9% + $0.30（¥2）
- 国际卡：+1.5%
- 货币转换：+1%

**优点：**
- 低费用
- 干净的结账体验
- 易于设置定期付款
- 自动发票
- 容易与预约系统集成

**缺点：**
- 学生不需要账户（直接刷卡）
- 不如PayPal知名
- 直接设置较技术化

**示例计算（¥300课程）：**
- 国内：¥300 - ¥8.70 - ¥2 = ¥289.30 收到
- 国际：¥300 - ¥13.20 - ¥2 = ¥284.80 收到

**适合人群：**
- 想要专业形象的导师
- 想与预约系统集成的人
- 销售定期付款/套餐的人

### 选项3：银行转账

**最便宜但最不方便**

直接银行转账费用最低，但对国际学生来说很复杂。

**费用：**
- 国内：免费或最低（¥3以下）
- 国际：SWIFT费用$15-50 + 中间银行费用

**优点：**
- 国内转账几乎免费
- 没有中间人
- 收到全额（国内）

**缺点：**
- 国际转账非常贵
- 转账确认需要时间（1-5天）
- 需要手动跟踪
- 退款复杂

**适合人群：**
- 只有国内学生的导师
- 课程费用高使费用比例低的情况
- 对技术不熟悉的学生

### 选项4：Wise（TransferWise）

**国际支付优化**

Wise专门从事国际汇款，提供实际汇率和低费用。

**费用：**
- 通常0.5-1%（取决于货币）
- 使用实际汇率（无额外加价）

**优点：**
- 国际费用非常低
- 实际汇率
- 可以有多币种账户
- 透明的费用结构

**缺点：**
- 国内转账不需要
- 学生需要Wise账户
- 与预约系统集成困难

**示例计算（$50课程）：**
- $50 - $0.50（1%）= $49.50 收到
- 比PayPal节省$1.50-2.00

**适合人群：**
- 有很多国际学生的导师
- 有欧洲、美国、英国学生的情况
- 对汇率敏感的人

### 选项5：集成平台（TutorLingua）

**预约 + 支付一站式**

像TutorLingua这样的一体化平台在一个地方处理预约和支付。

**费用：**
- 仅Stripe费用（2.9% + $0.30）
- 平台费用0%（对比Preply 18-33%）

**优点：**
- 预约时自动付款
- 学生管理集成
- 自动发票和收据
- [节省平台费用](/zh/blog/cluster-1-commissions/jiangdi-pingtai-shoufeijine)
- 套餐和定期付款简单

**缺点：**
- 月订阅费（$29-79）
- 绑定平台

**适合人群：**
- 认真的独立导师
- 从市场平台转换的人
- 想减少管理时间的人

## 费用比较（基于¥300课程）

| 方式 | 国内费用 | 国际费用 | 国内收到 | 国际收到 |
|------|---------|---------|---------|---------|
| 银行转账 | ~¥0 | ¥100+ | ¥300 | ¥200 |
| Stripe | ¥11 | ¥16 | ¥289 | ¥284 |
| PayPal | ¥13 | ¥16 | ¥287 | ¥284 |
| Wise | N/A | ¥3 | N/A | ¥297 |

## 按支付类型的策略

### 单次课程付款

**推荐：Stripe或PayPal**

让学生在预约时直接付款：
- 减少缺席
- 无需跟踪付款
- 专业体验

### 课程套餐

**推荐：Stripe + 预约系统集成**

预售5次、10次套餐：
- 稳定收入
- 增加学生承诺
- 可以提供每课折扣

示例：
- 单次课程：¥300
- 5次套餐：¥1,350（¥270/次，10%折扣）
- 10次套餐：¥2,400（¥240/次，20%折扣）

### 月度订阅

**推荐：Stripe定期付款**

为定期上课的学生提供月度订阅：
- 可预测的收入
- 提高学生留存率
- 自动付款（无需手动收费）

示例：
- 每周1次（每月4次）：¥1,080/月
- 每周2次（每月8次）：¥1,920/月

## 国际学生支付技巧

如果您有国际学生：

1. **提供多个选项**：Stripe + PayPal + Wise
2. **以USD/EUR显示价格**：避免汇率混乱
3. **添加汇率缓冲**：为汇率波动准备5-10%
4. **付款前确认货币**：明确以哪种货币收费

## 税务和记录管理

无论支付方式如何：

1. **记录所有交易**：电子表格或会计软件
2. **开具发票**：对所有付款
3. **保留收据**：用于报税
4. **季度审查**：分析收入和费用

**提示：** Stripe和PayPal自动提供交易历史，使记录管理变得容易。

## 专业发票制作

好的发票应包含：

- 您的姓名/商业名称
- 联系方式
- 发票编号（顺序）
- 日期
- 学生姓名
- 服务描述（例如："60分钟西班牙语课程"）
- 金额和货币
- 支付方式/说明
- 付款截止日期

**免费工具：**
- Wave（免费会计 + 发票）
- Invoice Ninja（免费）
- Stripe（自动发票）

## 支付政策设置

明确的政策预防问题：

### 取消政策
```
- 24小时以上提前取消：全额退款或改期
- 24小时内取消：收取50%
- 缺席：收取全额
```

### 退款政策
```
- 未使用的套餐：扣除手续费后退款
- 使用中的套餐：已使用课程按正价收费，其余退款
```

### 付款期限
```
- 单次课程：预约时预付
- 套餐：首次课程前付款
- 月度：每月1日自动收费
```

## 结论：简单开始，按需扩展

不要因为寻找完美的支付系统而推迟开始上课：

1. **现在**：用PayPal或银行转账开始
2. **5个学生以上**：设置Stripe
3. **销售套餐/订阅**：与预约系统集成
4. **从市场平台转换**：用TutorLingua整合一切

关键是**预付为默认**。如果课后才要求付款，您总是要追着要。预约 = 付款会让生活轻松很多。
