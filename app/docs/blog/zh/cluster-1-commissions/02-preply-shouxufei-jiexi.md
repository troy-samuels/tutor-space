## 引言

如果你在Preply上教课或正在考虑加入，了解佣金结构对于规划你的辅导收入至关重要。Preply的费用比简单的百分比更复杂——它们根据你的经验水平而变化，并包含一些让新教师措手不及的收费。

本指南详细解析2025年Preply向教师收取的费用，包含真实示例和计算，帮助你了解实际到手的薪酬。

---

## Preply佣金结构

### 试听课佣金：100%

这是让大多数新教师惊讶的部分：**Preply收取你与每个新学生第一节课的100%。**

没错——如果你收$30的试听课费用，你收到$0。Preply将全部金额作为连接你与学生的费用。

从Preply的角度来看：他们花钱做广告来获取那个学生。试听课佣金覆盖他们的客户获取成本。

从你的角度来看：你实际上在免费教一节课，以换取获得长期学生的*机会*。

### 持续佣金等级

试听课之后，Preply的佣金取决于你在平台上教了多少小时：

| 教学小时数 | 佣金费率 | 你的收入（$30/小时） |
|----------|--------|-------------------|
| 0-19小时 | 33% | $20.10 |
| 20-49小时 | 28% | $21.60 |
| 50-199小时 | 25% | $22.50 |
| 200-399小时 | 22% | $23.40 |
| 400+小时 | 18% | $24.60 |

### 达到每个等级需要多长时间

让我们对时间线保持现实：

- **20小时**（达到28%）：每周5-7小时约3-4周
- **50小时**（达到25%）：每周6小时约2个月
- **200小时**（达到22%）：每周6小时约8个月
- **400小时**（达到18%）：每周6小时约16个月

大多数兼职教师需要**1-2年**才能达到最低佣金等级。

## 实际收入计算

### 新教师场景（33%佣金）

**月度教学**：40节课，$25/小时

| 项目 | 金额 |
|-----|------|
| 总预约 | $1,000 |
| 试听课（假设4节） | -$100（你收到$0） |
| 常规课程（36 × $25） | $900总额 |
| 佣金（$900的33%） | -$297 |
| **净收入** | **$603** |
| **有效时薪** | **$15.08** |

你的标价是$25/小时，但你实际赚的是$15.08/小时。

### 资深教师场景（18%佣金）

**月度教学**：40节课，$35/小时

| 项目 | 金额 |
|-----|------|
| 总预约 | $1,400 |
| 试听课（假设2节） | -$70（你收到$0） |
| 常规课程（38 × $35） | $1,330总额 |
| 佣金（$1,330的18%） | -$239.40 |
| **净收入** | **$1,090.60** |
| **有效时薪** | **$27.27** |

即使在最佳等级，$35/小时的教师到手$27.27/小时。

## 佣金之外的隐藏成本

### 支付处理时间

Preply通过Payoneer、PayPal或Skrill支付教师。每个都有自己的费用：

- **Payoneer**：2%提现费 + 可能的货币转换
- **PayPal**：根据地区2-4%
- **Skrill**：1-2% + 可能的转换费

### "套餐折扣"影响

学生可以在Preply购买带内置折扣的课程套餐（通常5-10%）。这些折扣来自*你的*收入，而不是Preply的佣金。

示例：学生购买10节课套餐，5%折扣
- 正常费率：$30/小时 × 10 = $300
- 有折扣：$285（学生支付）
- Preply的25%佣金：$71.25
- **你的收入：$213.75**（$21.38/小时而不是$22.50）

### 取消政策

如果学生在少于4小时通知的情况下取消，你获得100%付款。但是，如果他们在政策窗口内取消，你什么都得不到——而且你已经锁定了那个时间段。

## 比较Preply与其他平台

| 平台 | 佣金 | 试听政策 | 最适合 |
|-----|------|---------|-------|
| Preply | 18-33% | 100%归平台 | 高流量，多语种 |
| iTalki | 15%固定 | 不要求免费试听 | 较低费用，灵活性 |
| Verbling | 15%固定 | 试听不同 | 专注视频质量 |
| Cambly | 固定费率（约$10.20/小时） | 不适用 | 简单，无需营销 |

Preply的佣金在主要平台中最高，但他们在学生获取方面投入巨大。这种权衡是否适合你，取决于你在其他地方吸引学生的能力。

## 最大化Preply收入的策略

### 1. 提高定价以补偿

如果你想作为新教师（33%佣金）赚取$25/小时净收入：
```
目标时薪 = $25 ÷ (1 - 0.33) = $37.31
```
将你的费率设为$38/小时以实际赚取你的目标。

### 2. 专注于学生留存

试听课实际上是免费营销。你的目标：将试听学生转化为长期常客。

- 为试听课做额外充分的准备
- 以清晰的未来课程计划结束
- 及时跟进课程笔记

### 3. 更快地教更多小时

佣金在每个等级显著下降。如果你能快速推进到50+小时，你将在每节未来的课上节省8%。

考虑暂时降低费率以预约更多课程并更快攀升等级——然后一旦达到25%或更低佣金就提高费率。

### 4. 考虑你的长期策略

Preply在建立学生群体方面非常出色。但一旦你有了稳定的回头客，每节课永远花费你18-33%。

许多教师战略性地使用Preply：
- 通过Preply获得新学生
- 随时间建立关系
- 为偏好直接预约的学生提供选项

## Preply值得吗？

### 如果你是以下情况，Preply有意义：
- 刚起步需要快速获得学生
- 不想做自己的营销
- 教授Preply需求高的语言
- 重视平台的基础设施（日程安排、支付、视频）

### 如果你是以下情况，考虑替代方案：
- 已经有学生基础
- 对基本营销感到舒适
- 想长期保留更多收入
- 教授Preply需求较低的语言

## 结论

Preply的佣金结构意味着：

- **新教师**失去常规课程的33% + 试听课的100%
- **资深教师**（400+小时）仍然失去18% + 试听课
- **实际时薪**明显低于标价
- **套餐折扣**进一步降低你的收入

对于学生获取，这些成本可能是值得的。对于长期回头客，你在为不再需要的服务支付溢价。

理解这些数字帮助你做出关于定价、平台组合以及何时开发替代预约渠道的明智决策。
