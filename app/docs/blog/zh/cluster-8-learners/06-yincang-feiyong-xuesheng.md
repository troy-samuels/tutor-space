## 要点摘要

- 家教平台向**教师和学生双方**收费——不是只收一方
- 学生费用从**每节课0到15+美元**不等，取决于平台
- **Preply** 从教师抽取高达33%，**还**向学生收取额外费用
- **iTalki** 更透明，但仍有货币转换成本
- **总平台开销**可达你支付金额的35-50%
- **直接预订**为双方消除大部分费用

---

## 引言：你支付的比你想象的多

当你在线预订语言教师时，你看到一个价格——比如40美元/小时。你以为这就是你支付的，也是教师收到的，对吧？

**错了。**

家教平台采用双重收费模式：他们向教师收取佣金，**同时**往往向学生收取额外费用。结果？你支付的很大一部分给了平台，而不是你的教师。

本指南详细分析主要平台向学生收取的费用，以及这些隐藏费用如何增加你的实际学习成本。

---

## 双边收费模式

### 平台如何赚钱

大多数家教平台从交易双方获利：

**教师端**：每节课15-33%的佣金
**学生端**：服务费、平台费、货币转换

当你支付40美元一节课时，分配可能是这样的：

| 你的40美元去向 | 金额 |
|--------------|------|
| 给你的教师 | $26-34 |
| 给平台（教师佣金） | $6-13 |
| 给平台（学生费用） | $0-6 |
| 支付处理 | $1-2 |

**结果**：你花的每一美元，教师可能只收到65-85分。

*想了解教师的角度？我们的[详细佣金对比](/zh/blog/preply-italki-verbling-youhui-bijiao-2025)展示了每个平台到底从教师那里收取多少。*

## 平台费用详解

### Preply：最激进的收费结构

**教师佣金**：18-33%（取决于教学时长）

**学生端费用**：
- 服务费：可变，通常在显示价格基础上增加10-18%
- 首课费：有时单独收取
- 货币转换：如果你的货币不同会有额外加价

**计算示例**：

你看到一个教师标价35美元/小时。你预订后实际支付：

| 组成部分 | 费用 |
|---------|------|
| 教师基础费率 | $35.00 |
| 服务费（约15%） | $5.25 |
| **你支付** | **$40.25** |
| 平台从教师扣除（33%） | $11.55 |
| **教师收到** | **$23.45** |

**你的$40.25课程 = $23.45给教师 + $16.80给平台**

这是**42%给了平台**。

### iTalki：更透明，但仍有成本

**教师佣金**：固定15%

**学生端费用**：
- 无明确服务费（费用已计入定价）
- 货币转换：使用非美元货币时2-3%
- 支付方式费用：某些方式有额外成本

**计算示例**：

你看到一个教师40美元/小时：

| 组成部分 | 费用 |
|---------|------|
| 显示价格 | $40.00 |
| 货币转换（约2.5%） | $1.00 |
| **你支付** | **$41.00** |
| 平台从教师扣除（15%） | $6.00 |
| **教师收到** | **$34.00** |

**你的$41课程 = $34给教师 + $7给平台**

这是**17%给了平台**——比Preply好，但仍然可观。

### Verbling：与iTalki类似

**教师佣金**：约15%

**学生端费用**：
- 通常无明确服务费
- 有货币转换成本
- 支付处理费用各异

Verbling对教师更友好，但仍从教师端抽取。

### Cambly：固定定价，透明度较低

**模式**：学生向Cambly付费；Cambly按固定费率支付教师

**学生成本**：
- 订阅制定价
- 教师选择有限
- 费率看起来实惠但教师报酬相对较低

**问题**：教师每分钟赚$0.17-$0.20（约$10-12/小时），而学生可能支付相当于$15-30/小时。平台获取巨大差价。

## 货币转换税

如果你不在美国，货币转换是每节课的隐藏税。

### 如何运作

1. 你用本地货币支付（欧元、英镑、澳元等）
2. 平台按他们的汇率转换（不是市场汇率）
3. 你比中间市场汇率多付2-5%
4. 教师提现时可能面临另一次转换

### 实际成本示例

**市场汇率**：1美元 = 0.85欧元
**平台汇率**：1美元 = 0.82欧元（3.5%加价）

一节40美元的课程：
- 按市场汇率：€34.00
- 按平台汇率：€35.19
- **隐藏费用**：每节课€1.19

一年50节课：**仅货币隐藏费用就是€59.50**。

## 订阅陷阱和积分系统

### 平台积分：旨在锁定你

许多平台使用积分系统：

1. 你购买积分（通常有小"折扣"）
2. 未使用的积分会过期
3. 积分无法转让或退款
4. 你被锁定在该平台上

**示例**：花$90购买100积分（10%折扣！）。但是：
- 积分6个月后过期
- 你只用了80积分
- 未使用的20积分 = $18损失
- 你的实际折扣：0%

### 订阅自动续费

一些平台提供的订阅：
- 没有清晰通知就自动续费
- 取消窗口你可能会错过
- 在你注意到之前就收取下月费用
- 退款困难

## 转嫁给你的"教师税"

### 教师佣金如何影响你的价格

当平台从教师抽取15-33%时，教师必须提高价格以维持收入。

**无平台费用**：教师可能收费40美元/小时并赚40美元
**有25%平台费用**：同一教师必须收费54美元/小时才能赚40美元

**因为有平台参与，你为同一教师多付35%**。

### 市场扭曲

平台创造人为的价格底线：
- 本来收30美元/小时的教师必须收40+美元才能生存
- 预算友好的教师变成中档
- 中档教师变成高端
- 所有人都付更多

## 案例研究：50节课的真实成本

让我们比较一下，对于一个想要每小时赚35美元的教师，你上50节课会支付多少：

### 场景A：Preply（高费用平台）

| 组成部分 | 每节课 | 50节课 |
|---------|--------|--------|
| 教师需要赚 | $35.00 | $1,750 |
| Preply佣金（25%） | +$11.67 | +$583 |
| 服务费（约15%） | +$7.00 | +$350 |
| **你支付** | **$53.67** | **$2,683** |

### 场景B：iTalki（较低费用平台）

| 组成部分 | 每节课 | 50节课 |
|---------|--------|--------|
| 教师需要赚 | $35.00 | $1,750 |
| iTalki佣金（15%） | +$6.18 | +$309 |
| 货币转换（约2%） | +$0.82 | +$41 |
| **你支付** | **$42.00** | **$2,100** |

### 场景C：直接预订

| 组成部分 | 每节课 | 50节课 |
|---------|--------|--------|
| 教师需要赚 | $35.00 | $1,750 |
| 支付处理（约3%） | +$1.08 | +$54 |
| **你支付** | **$36.08** | **$1,804** |

### 差异

| 平台 | 50节课成本 | 相对直接预订溢价 |
|-----|-----------|-----------------|
| Preply | $2,683 | +$879（+49%） |
| iTalki | $2,100 | +$296（+16%） |
| 直接 | $1,804 | 基准 |

**使用Preply而非直接预订，同样50节课多花$879。**

## 如何避免隐藏费用

### 1. 预订前计算真实成本

承诺之前，计算：
- 显示价格 + 任何服务费
- 如适用的货币转换
- 积分系统陷阱
- 与直接预订等价物比较

### 2. 用平台发现，然后直接预订

聪明的策略：
1. 在平台上找到好教师（他们擅长发现）
2. 上几节课验证质量
3. 询问直接预订选项
4. 持续课程转为直接预订

### 3. 直接问你的教师

上了几节课后，简单地问：
> "我想长期继续和你学习。你有任何直接预订的方式吗？这对我们双方在经济上都更好。"

大多数教师会感激这个——他们赚更多并保留承诺的学生。

### 4. 用平台的基础货币支付

如果你必须使用平台：
- 如果那是他们的基础货币，用美元支付
- 使用无外币交易费的卡
- 尽可能避免平台货币转换

## 常见问题

### 为什么平台不显示真实总成本？

较低的显示价格获得更多点击。平台在结账时添加费用，那时你已经打算预订了。

### 为了保护值得支付平台费用吗？

对于与未知教师的第一节课，也许。对于与信任的教师持续上课？不——你在为不再需要的服务付费。

### 我能退还平台费用吗？

通常不能。服务费在大多数平台上不可退款。积分可能有复杂的退款政策。

### 所有平台都向教师和学生双方收费吗？

大多数以某种形式收费。即使"只收佣金"的平台也会抬高教师价格，学生最终付费。

### 为什么不是更多教师提供直接预订？

许多确实提供，但是悄悄地——他们不能在平台上宣传。你必须主动问。

---

## 结论：了解你真正在支付什么

家教平台提供价值——它们帮助你发现教师并处理初始信任问题。但这种价值不能证明无限期地为每节课多付20-50%是合理的。

**你的钱，你的选择：**
- 用平台发现（它们擅长这个）
- 了解真实成本（不仅是显示价格）
- 与你信任的教师转为直接预订
- 每年节省数百或数千元

你喜欢的教师不会因为你如何预订而改变。但你支付多少——以及他们赚多少——绝对会。
