## Pontos-chave

- **Não existe a plataforma perfeita** – mas a certa para sua situação
- **Os marketplaces** (Preply, iTalki, Verbling) são ideais para adquirir alunos
- **A reserva direta** (TutorLingua, site próprio) é ideal para alunos recorrentes
- **As plataformas de nicho** (Cambly, Lingoda) têm prós e contras específicos
- **O modelo híbrido** é ótimo para a maioria dos tutores

---

## Por Que Buscar Alternativas?

### Motivos Comuns de Insatisfação

1. **Comissões altas** (15-33% nas grandes plataformas)
2. **Aulas experimentais não pagas** (especialmente no Preply)
3. **Dependência do algoritmo** (visibilidade fora de controle)
4. **Liberdade de preço limitada**
5. **Falta de relacionamento com o aluno** (a plataforma "possui" o contato)

### O Que Você Busca

- Comissões mais baixas
- Mais controle sobre preços e relacionamentos
- Ganhos mais estáveis
- Aparência mais profissional

## Plataformas de Nicho e Especializadas

### Cambly

| Aspecto | Avaliação |

| **Flexibilidade** | Muito alta (conversas instantâneas) |

**Desvantagens:**

- Nenhuma fidelização de alunos

- Estrutura fixa
- Horas regulares

**Desvantagens:**
- Taxa horária baixa
- Nenhuma personalização
- Regras rígidas

**Conclusão:** Para professores que preferem estrutura e não querem criar materiais.

|---------|-----------|

**Vantagens:**
- Forte mercado EUA
- Muitas disciplinas
- Preços altos possíveis

**Desvantagens:**
- Comissão alta
- Apenas alunos EUA relevantes
- Forte concorrência

**Conclusão:** Bom para tutores nos EUA com espectro de matérias mais amplo.

### Superprof

| Aspecto | Avaliação |
|---------|-----------|
| **Modelo** | Assinatura mensal para tutores |
| **Base de alunos** | Europa (especialmente França, Alemanha) |
| **Para quem** | Aulas locais + online |

**Vantagens:**
- Nenhuma comissão por aula
- Alunos locais e online
- Espectro de matérias amplo

**Desvantagens:**
- Mensalidade mesmo sem reservas
- Qualidade de leads variável
- Menos focado em idiomas

**Conclusão:** Interessante para tutores que também ensinam localmente.

## A Alternativa da Reserva Direta

### O Que É Reserva Direta?

Em vez de passar por um marketplace, você reserva alunos através de:
- Sua página de reservas (ex: TutorLingua)
- Seu site próprio
- Canais de redes sociais
- Indicações

**A diferença:** 0% de comissão nas suas aulas.

### TutorLingua

| Aspecto | Avaliação |
|---------|-----------|
| **Comissão** | 0% |
| **Custos** | Início gratuito, planos Pro disponíveis |
| **Base de alunos** | Você traz seus alunos |
| **Para quem** | Tutores com alunos recorrentes ou marketing próprio |

**Vantagens:**
- 0% comissão em todas as aulas
- Página de reservas profissional
- Pagamentos integrados (Stripe)
- Sincronização de calendário
- Gestão de alunos (CRM)
- Lembretes automáticos
- Construído para tutores (não genérico)

**Desvantagens:**
- Você precisa encontrar seus alunos
- Nenhum tráfego de marketplace integrado

**Ideal para:**
- Alunos recorrentes dos marketplaces
- Indicações e boca a boca
- Seguidores nas redes sociais
- Alunos que te encontram no Google

### Calendly + Stripe (solução faça-você-mesmo)

| Aspecto | Avaliação |
|---------|-----------|
| **Comissão** | ~3% (taxa Stripe) |
| **Custos** | Calendly Free ou Pro (8-12€/mês) |
| **Para quem** | Tutores experientes em tecnologia |

**Vantagens:**
- Ferramentas conhecidas
- Alta flexibilidade
- Econômico

**Desvantagens:**
- Combinar várias ferramentas
- Não otimizado para tutoria
- Nenhuma gestão de alunos

### Site Próprio + WooCommerce

| Aspecto | Avaliação |
|---------|-----------|
| **Comissão** | ~3% (taxa de pagamento) |
| **Custos** | Hospedagem (~10€/mês) + tempo |
| **Para quem** | Tutores com conhecimento de web design |

**Vantagens:**
- Máximo controle
- Marca própria
- Potencial SEO

**Desvantagens:**
- Alto investimento de tempo
- Manutenção técnica
- Não para iniciantes

## A Estratégia Ótima: Modelo Híbrido

### Por Que Híbrido?

Nenhuma plataforma única satisfaz todas as necessidades:
- **Marketplace** = bom para novos clientes, caro para recorrentes
- **Reserva direta** = econômica, mas você precisa encontrar seus alunos

**A solução**: Use ambos estrategicamente.

### O Setup Recomendado

| Canal | Propósito | Quota |
|-------|-----------|-------|
| **Preply ou iTalki** | Adquirir novos alunos | 30-40% |
| **TutorLingua** | Atender recorrentes | 50-60% |
| **Redes sociais** | Construção de marca, pedidos diretos | 10-20% |

### Como Funciona

1. **Descoberta**: Aluno te encontra no Preply/iTalki
2. **Relacionamento**: Após 3-6 meses a confiança está construída
3. **Transição**: Aluno reserva voluntariamente direto
4. **Resultado**: 0% em vez de 15-33% comissão para recorrentes

## Guia de Decisão: Qual Alternativa É Para Você?

### Se Você Está Começando

**Recomendação:** iTalki ou Preply
- Você precisa de tráfego de alunos
- As comissões são aceitáveis para fase de construção
- Aprenda o que funciona

### Se Você Está Estabelecido (1+ ano)

**Recomendação:** Modelo híbrido (Marketplace + TutorLingua)
- Novos alunos via marketplace
- Recorrentes via reserva direta
- Aumente gradualmente a quota direta

### Se Você Tem Grande Seguimento

**Recomendação:** TutorLingua ou site próprio
- Você não precisa de tráfego de marketplace
- Maximize ganhos com 0% comissão
- Controle total sobre preços e marca

### Se Você Busca Renda Extra

**Recomendação:** Cambly ou Lingoda
- Horários flexíveis
- Nenhum marketing necessário
- Aceite taxa horária mais baixa

### Se Você É Professor Certificado

**Recomendação:** Verbling + reserva direta
- Menos concorrência no Verbling
- Preços mais altos possíveis
- Alunos atentos à qualidade

## Perguntas Frequentes

### Posso usar várias plataformas ao mesmo tempo?

Sim! Nenhuma das grandes plataformas exige exclusividade. Só tome cuidado com sobreposições de calendário.

### Como passo do Preply para reserva direta?

1. Configure infraestrutura para reserva direta (TutorLingua ou similar)
2. Construa presença fora da plataforma (redes sociais)
3. Espere oportunidades naturais (alunos que perguntam)
4. Ofereça a alternativa (fora do chat do Preply!)

### A reserva direta é legal?

Sim, absolutamente. Você só não pode recrutar dentro dos chats dos marketplaces. Página de reservas própria e contatos externos são legais.

### Qual plataforma tem mais alunos brasileiros/portugueses?

Para quem aprende português: Preply e iTalki são ambas fortes. Para aulas locais brasileiras/portuguesas: Superprof.

### Preciso de um site próprio?

Não necessariamente. TutorLingua oferece uma página de reservas profissional sem conhecimento de web design. Um site próprio é opcional para branding/SEO avançado.
