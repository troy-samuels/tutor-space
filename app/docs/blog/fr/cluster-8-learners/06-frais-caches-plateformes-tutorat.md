## Points Clés (Résumé)

- **Les plateformes prennent 15-33%** de chaque cours en commission
- **Ces frais sont intégrés dans le prix** que vous payez (pas visibles séparément)
- **Les tuteurs augmentent leurs tarifs** pour compenser les commissions
- **Les frais de paiement** peuvent ajouter 2-5% supplémentaires
- **La réservation directe élimine** la majorité de ces frais cachés

---

## Introduction : Ce Que Vous Ne Voyez Pas

Quand vous réservez un cours à 40€ sur Preply, vous pensez probablement que votre tuteur reçoit 40€.

**Faux.**

Votre tuteur reçoit entre 27€ et 33€. La différence ? Elle va à la plateforme.

Ce guide révèle exactement où va votre argent et comment en garder plus dans votre poche (et celle de votre tuteur).

---

## Comment Les Plateformes Gagnent Leur Argent

### Le Modèle De Commission

Les plateformes de tutorat fonctionnent sur un modèle de commission :

| Plateforme | Commission | Ce que le tuteur garde |
|------------|------------|------------------------|
| Preply | 18-33% | 67-82% |
| iTalki | 15% | 85% |
| Verbling | ~15% | ~85% |
| Superprof | 0% (frais étudiant) | 100% |

### Ce Que Cela Signifie Concrètement

Sur un cours affiché à **40€** :

**Preply (premier cours, 33%)** :
- Vous payez : 40€
- Tuteur reçoit : 26,80€
- Plateforme garde : 13,20€

**iTalki (15%)** :
- Vous payez : 40€
- Tuteur reçoit : 34€
- Plateforme garde : 6€

## Les Frais Cachés Au-Delà De La Commission

### 1. Frais De Conversion De Devise

Si vous payez dans une devise différente de celle du tuteur :
- **Frais de conversion** : 2-4%
- Souvent intégrés dans un taux de change défavorable

### 2. Frais De Paiement

Certaines méthodes de paiement ajoutent des frais :
- Carte de crédit : 0-3%
- PayPal : 2-4%
- Virement international : frais fixes

### 3. Frais D'Abonnement (Certaines Plateformes)

Quelques plateformes facturent :
- Abonnement mensuel premium
- Accès à certaines fonctionnalités
- Réservation prioritaire

### 4. Le Coût Caché : Prix Gonflés

C'est le plus gros frais caché : **les tuteurs augmentent leurs prix pour compenser les commissions**.

**Exemple** :
Un tuteur veut gagner 35€/heure.
- Sur Preply (25% commission) : doit afficher **47€/heure**
- En direct : peut afficher **35€/heure**

**Vous payez 12€ de plus** pour le même cours.

---

## Analyse Détaillée Par Plateforme

### Preply

**Structure de commission** :
- 33% pour les 20 premières heures avec chaque étudiant
- Diminue progressivement jusqu'à 18% après 200+ heures

**Frais supplémentaires** :
- Frais de remboursement si vous changez d'avis
- Certaines promotions cachent des conditions

**Impact sur vous** :
Les tuteurs débutants sur Preply (avec 33% de commission) facturent plus cher pour compenser.

### iTalki

**Structure de commission** :
- 15% fixe pour tous les tuteurs

**Frais supplémentaires** :
- Frais de retrait pour les tuteurs (PayPal, Payoneer)
- Ces frais sont parfois répercutés sur les prix

**Impact sur vous** :
Plus prévisible que Preply, mais 15% reste significatif.

### Verbling

**Structure de commission** :
- ~15% (similaire à iTalki)

**Impact sur vous** :
Base d'utilisateurs plus petite = moins de choix de tuteurs.

### Superprof

**Modèle différent** :
- Pas de commission sur les cours
- Frais de "mise en relation" payé par l'étudiant
- Généralement un paiement unique

**Impact sur vous** :
Moins cher à long terme si vous gardez le même tuteur.

## Calcul : Combien Perdez-Vous Vraiment ?

### Scénario : 4 Cours Par Mois Pendant 1 An

**Sur Preply (prix gonflé de 47€ vs prix direct de 35€)** :
- Vous payez : 47€ × 4 × 12 = **2256€/an**
- Tuteur reçoit (après 25% commission moyenne) : ~1692€

**En réservation directe (35€)** :
- Vous payez : 35€ × 4 × 12 = **1680€/an**
- Tuteur reçoit (après ~2% Stripe) : ~1646€

**Résultat** :
- Vous économisez : **576€/an**
- Tuteur gagne presque autant (1646€ vs 1692€)
- Plateforme perd : ~564€

### Scénario : Étudiant Intensif (8 Cours/Mois)

**Sur plateforme** : 4512€/an
**En direct** : 3360€/an
**Économie** : **1152€/an**

## Pourquoi Les Plateformes Facturent Ces Frais

### Services Fournis

Les plateformes fournissent réellement des services :

✅ **Marketing** : Elles trouvent des étudiants pour les tuteurs
✅ **Infrastructure** : Plateforme de réservation, paiements
✅ **Confiance** : Avis, vérifications, garanties
✅ **Support** : Service client, résolution de conflits

### Le Problème

Ces services sont **utiles pour le premier cours**, pas pour le 50ème.

- Marie trouve son tuteur via Preply → La plateforme a apporté de la valeur
- Marie prend son 50ème cours → La plateforme prend encore 20%+ pour... quoi ?

C'est là que le modèle devient discutable.

## Comment Économiser Sur Vos Cours

### Option 1 : Réserver Directement (Recommandé)

Demandez à votre tuteur s'il offre des réservations directes :
- Prix souvent 10-20% moins cher
- Le tuteur garde plus
- Vous gardez plus
- Tout le monde gagne (sauf la plateforme)

### Option 2 : Négocier Des Packs

Beaucoup de tuteurs offrent des réductions pour les engagements :
- Pack 10 cours : -10%
- Pack 20 cours : -15%
- Abonnement mensuel : -20%

### Option 3 : Tuteurs Communautaires

Sur iTalki, les "tuteurs communautaires" (non certifiés) sont moins chers :
- Prix souvent 50% inférieurs
- Qualité variable mais parfois excellent
- Idéal pour la conversation

### Option 4 : Comparer Les Plateformes

Un même tuteur peut être sur plusieurs plateformes avec des prix différents :
- Vérifiez iTalki ET Preply
- Cherchez leur site personnel
- Demandez s'ils ont une page de réservation directe

## Ce Que Vous Pouvez Faire

### Étape 1 : Calculez Vos Frais Actuels

Regardez vos factures des derniers mois :
- Combien avez-vous dépensé ?
- À quel tarif horaire ?
- Combien de cours ?

### Étape 2 : Parlez À Votre Tuteur

Posez la question simplement :
> "Proposez-vous des réservations en dehors de [plateforme] ?"

### Étape 3 : Comparez Les Options

Demandez le tarif direct et comparez :
- Prix plateforme vs prix direct
- Économie potentielle annuelle
- Services perdus (remboursement plateforme ?)

### Étape 4 : Faites Le Choix Éclairé

Si vous faites confiance à votre tuteur et que les économies sont significatives, la réservation directe fait sens.

## Questions Fréquentes

### Est-ce que les plateformes vont me pénaliser ?

Non. Vous êtes libre de réserver où vous voulez. Les plateformes ne peuvent pas vous empêcher de trouver un tuteur sur Instagram ou via une recherche Google.

### Mon tuteur risque-t-il des problèmes ?

Les tuteurs ne peuvent pas solliciter directement les étudiants SUR la plateforme. Mais si vous les contactez via d'autres canaux (leur site, réseaux sociaux), c'est parfaitement légal.

### Et si j'ai un problème avec un cours réservé en direct ?

Discutez des politiques de remboursement avant de réserver. Les tuteurs sérieux ont des politiques claires. Les paiements via Stripe offrent une protection acheteur.

### Comment savoir si le tuteur est fiable pour la réservation directe ?

- Vous avez déjà eu plusieurs cours ensemble
- Il utilise un système de paiement professionnel
- Il a des avis positifs sur les plateformes
- Il a un site web ou une présence professionnelle

## Conclusion : Soyez Un Consommateur Informé

Les plateformes de tutorat ne sont pas "mauvaises"—elles fournissent un service réel. Mais elles facturent ce service de manière continue, même quand leur valeur ajoutée diminue.

**En tant qu'étudiant informé**, vous pouvez :
- Utiliser les plateformes pour **découvrir** des tuteurs
- Passer à la réservation directe pour les **cours répétés**
- **Économiser des centaines d'euros** par an
- **Mieux rémunérer** votre tuteur

C'est gagnant-gagnant pour vous et votre tuteur. Seule la plateforme y perd—et elle survivra.
