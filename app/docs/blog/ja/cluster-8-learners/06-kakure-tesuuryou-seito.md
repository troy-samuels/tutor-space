## 要約（TL;DR）

- 講師プラットフォームは**講師と生徒の両方**に課金—どちらか一方ではない
- 生徒の手数料は**1レッスンあたり0円〜1,500円以上**（プラットフォームにより異なる）
- **Preply**は講師から最大33%を取り、生徒には追加手数料も
- **iTalki**はより透明だが、為替換算コストがある
- **総プラットフォーム経費**は支払額の35〜50%に達することも
- **直接予約**で双方の手数料のほとんどを排除

---

## はじめに：思っている以上に払っている

オンラインで語学講師を予約する時、価格を見ます—例えば時給4,000円。それがあなたの支払額で、講師の受取額だと思いますよね？

**間違いです。**

講師プラットフォームは二重取りモデルで運営しています：講師に手数料を課金し、生徒にも追加手数料を課金することが多い。結果？あなたの支払いのかなりの部分がプラットフォームに行き、講師には行きません。

このガイドでは、主要プラットフォームが生徒に何を課金し、これらの隠れた手数料が実際の学習コストをどう膨らませるか詳しく解説します。

---

## 二重手数料モデル

### プラットフォームの稼ぎ方

ほとんどの講師マーケットプレイスは取引の両側から利益を得ます：

**講師側**：すべてのレッスンの15〜33%の手数料
**生徒側**：サービス料、プラットフォーム料、為替換算

4,000円のレッスンを払う時、内訳はこうなりかねない：

| あなたの4,000円の行き先 | 金額 |
|------------------------|------|
| 講師へ | 2,600〜3,400円 |
| プラットフォーム（講師手数料） | 600〜1,300円 |
| プラットフォーム（生徒手数料） | 0〜600円 |
| 決済処理 | 100〜200円 |

**結果**：あなたの支払いの65〜85%しか講師に届かないことも。

*講師の視点に興味がありますか？[詳細な手数料比較](/ja/blog/preply-italki-verbling-commission-hikaku-2025)で各プラットフォームが講師からいくら取るか確認できます。*

## プラットフォーム別の内訳

### Preply：最も積極的な手数料体系

**講師手数料**：18〜33%（教えた時間により異なる）

**生徒側の手数料**：
- サービス料：変動、通常表示価格の10〜18%が追加
- 初回レッスン料：別途請求されることも
- 為替換算：通貨が異なる場合、追加マークアップ

**計算例**：

時給3,500円の講師を見つけました。予約すると実際に払うのは：

| 項目 | 金額 |
|------|------|
| 基本講師料金 | 3,500円 |
| サービス料（約15%） | 525円 |
| **あなたの支払い** | **4,025円** |
| 講師への手数料（33%） | 1,155円 |
| **講師の受取** | **2,345円** |

**あなたの4,025円のレッスン = 講師に2,345円 + プラットフォームに1,680円**

これは**42%がプラットフォームへ**。

### iTalki：より透明だが、コストはある

**講師手数料**：一律15%

**生徒側の手数料**：
- 明示的なサービス料なし（手数料は価格設定に含まれる）
- 為替換算：非USD通貨なら2〜3%
- 決済方法手数料：一部の方法は追加コスト

**計算例**：

時給4,000円の講師を見つけた場合：

| 項目 | 金額 |
|------|------|
| 表示価格 | 4,000円 |
| 為替換算（約2.5%） | 100円 |
| **あなたの支払い** | **4,100円** |
| 講師への手数料（15%） | 600円 |
| **講師の受取** | **3,400円** |

**あなたの4,100円のレッスン = 講師に3,400円 + プラットフォームに700円**

これは**17%がプラットフォームへ**—Preplyよりましだが、それでも大きい。

### Verbling：iTalkiと同様

**講師手数料**：約15%

**生徒側の手数料**：
- 一般的に明示的なサービス料なし
- 為替換算コストあり
- 決済処理は様々

Verblingは講師寄りだが、講師側からは取り分を取る。

### Cambly：固定価格、透明性が低い

**モデル**：生徒はCamblyに支払い、Camblyが講師に固定レートを支払う

**生徒のコスト**：
- サブスクリプションベースの価格設定
- 講師の選択肢が限定的
- 料金は手頃に見えるが、講師は比較的低い報酬

**問題**：講師は分あたり17〜20円程度（時給約1,000〜1,200円相当）を稼ぐ一方、生徒は時給換算1,500〜3,000円を払うことも。プラットフォームが大きなマージンを取る。

## 為替換算という税金

日本以外にいるなら、為替換算はすべてのレッスンにかかる隠れた税金です。

### 仕組み

1. あなたの現地通貨（円など）で払う
2. プラットフォームが自社レートで換算（市場レートではない）
3. 中間市場為替レートより2〜5%高く払う
4. 講師も引き出し時に別の換算を受けることも

### 実際のコスト例

**市場為替レート**：1 USD = 150円
**プラットフォームレート**：1 USD = 145円（3.5%マークアップ）

40ドルのレッスンで：
- 市場レートなら：6,000円
- プラットフォームレートで：6,210円
- **隠れた手数料**：210円/レッスン

年間50レッスンで：**為替の隠れた手数料だけで10,500円**。

---

## サブスクリプションの罠とクレジットシステム

### プラットフォームクレジット：囲い込みの設計

多くのプラットフォームはクレジットシステムを使用：

1. クレジットを購入（わずかな「割引」付き）
2. クレジットは未使用で期限切れ
3. クレジットは譲渡不可、返金不可
4. そのプラットフォームに縛られる

**例**：100クレジットを9,000円で購入（10%割引！）。しかし：
- クレジットは6ヶ月で期限切れ
- 80クレジットしか使わなかった
- 未使用の20クレジット = 1,800円の損失
- 実際の割引：0%

### サブスクリプションの自動更新

一部のプラットフォームはサブスクリプションを提供し：
- 明確な通知なしで自動更新
- キャンセル期間を逃すかも
- 気づく前に翌月分が課金
- 返金が難しい

## あなたに転嫁される「講師税」

### 講師手数料が価格に影響する仕組み

プラットフォームが講師から15〜33%を取ると、講師は収入を維持するために価格を上げなければなりません。

**プラットフォーム手数料なしの場合**：講師は時給4,000円を設定して4,000円を稼ぐ
**25%のプラットフォーム手数料ありの場合**：同じ講師は4,000円稼ぐために5,400円/時間を設定する必要あり

**同じ講師に35%以上多く払う**ことになる、プラットフォームが関わるだけで。

### 市場の歪み

プラットフォームは人為的な価格下限を作る：
- 時給3,000円を請求したい講師は生き残るために4,000円以上を請求する必要
- 予算に優しい講師が中価格帯に
- 中価格帯の講師がプレミアムに
- みんながより多く払う

## ケーススタディ：50レッスンの本当のコスト

時給3,500円稼ぎたい講師との50レッスンを比較：

### シナリオA：Preply（高手数料プラットフォーム）

| 項目 | 1レッスン | 50レッスン |
|------|---------|----------|
| 講師が稼ぐ必要 | 3,500円 | 175,000円 |
| Preply手数料（25%） | +1,167円 | +58,350円 |
| サービス料（約15%） | +700円 | +35,000円 |
| **あなたの支払い** | **5,367円** | **268,350円** |

### シナリオB：iTalki（低手数料プラットフォーム）

| 項目 | 1レッスン | 50レッスン |
|------|---------|----------|
| 講師が稼ぐ必要 | 3,500円 | 175,000円 |
| iTalki手数料（15%） | +618円 | +30,900円 |
| 為替換算（約2%） | +82円 | +4,100円 |
| **あなたの支払い** | **4,200円** | **210,000円** |

### シナリオC：直接予約

| 項目 | 1レッスン | 50レッスン |
|------|---------|----------|
| 講師が稼ぐ必要 | 3,500円 | 175,000円 |
| 決済処理（約3%） | +108円 | +5,400円 |
| **あなたの支払い** | **3,608円** | **180,400円** |

### 差額

| プラットフォーム | 50レッスン費用 | 直接比でプレミアム |
|----------------|---------------|-------------------|
| Preply | 268,350円 | +87,950円（+49%） |
| iTalki | 210,000円 | +29,600円（+16%） |
| 直接 | 180,400円 | 基準 |

**同じ50レッスンで、Preplyの代わりに直接予約すると87,950円多く払う。**

## 隠れた手数料を避ける方法

### 1. 予約前に本当のコストを計算

コミットする前に計算：
- 表示価格 + サービス料
- 該当すれば為替換算
- クレジットシステムの罠
- 直接予約との比較

### 2. プラットフォームは発見に使い、その後は直接予約

賢い戦略：
1. マーケットプレイスで良い講師を見つける（発見は得意）
2. 数回レッスンを受けて質を確認
3. 直接予約オプションについて聞く
4. 継続レッスンは直接に移行

### 3. 講師に直接聞く

数回のレッスン後、単純に聞く：
> 「長期的に続けたいと思っています。直接予約する方法はありますか？お互いにとって経済的にメリットがありますよね。」

多くの講師は感謝します—収入が増え、コミットした生徒を維持できます。

### 4. プラットフォームの基本通貨で支払う

プラットフォームを使う必要があるなら：
- 基本通貨がUSDならUSDで支払う
- 外国取引手数料なしのカードを使う
- 可能ならプラットフォームの為替換算を避ける

## よくある質問

### なぜプラットフォームは本当の総コストを表示しない？

低い表示価格の方がクリックされる。プラットフォームはすでに予約にコミットしたチェックアウト時に手数料を追加。

### 保護のためにプラットフォーム手数料を払う価値はある？

未知の講師との初回レッスンなら、たぶん。信頼できる講師との継続レッスンには？もう必要ないサービスに払っていることになる。

### プラットフォーム手数料の返金は受けられる？

通常は無理。サービス料はほとんどのプラットフォームで返金不可。クレジットには複雑な返金ポリシーがあることも。

### すべてのプラットフォームが講師と生徒の両方に課金する？

ほとんどが何らかの形でそうしています。「手数料のみ」のプラットフォームでも講師の価格を膨らませ、結局生徒が払います。

### もっと多くの講師が直接予約を提供しないのはなぜ？

多くは提供していますが、静かに—マーケットプレイス上では宣伝できません。聞く必要があります。

---

## 結論：本当に払っている金額を知る

講師プラットフォームは価値を提供します—講師を発見し、最初の信頼を処理するのに役立ちます。でもその価値は、永遠にすべてのレッスンに20〜50%多く払うことを正当化しません。

**あなたのお金、あなたの選択：**
- 発見にはプラットフォームを使う（それは得意）
- 本当のコストを理解する（表示価格だけでなく）
- 信頼できる講師とは直接予約に移行
- 年間数万円を節約

好きな講師は、予約方法で変わりません。でも支払う金額—そして彼らが稼ぐ金額—は確実に変わります。
