## はじめに

Preplyで1時間3,500円を請求しています。でも実際にはいくら*稼いで*いますか？

手数料、体験レッスン、決済処理、その他の費用を差し引くと、実際の時給は思っているよりずっと低いかもしれません。この数字を理解することは、価格設定の決定と財務計画に不可欠です。

実際の収入を計算しましょう。

---

## 実際の時給の公式

実際の時給は表示価格ではありません。これです：

```
実際の時給 = (総収入 - すべての手数料) ÷ 総労働時間
```

「総労働時間」にはレッスン時間に加えて管理時間（スケジューリング、準備、コミュニケーション）が含まれます。

## ステップ1：プラットフォーム手数料を計算

### Preplyのティア別手数料

| 教授時間 | 手数料 | 3,500円レッスンの受取額 |
|---------|-------|---------------------|
| 0-19 | 33% | 2,345円 |
| 20-49 | 28% | 2,520円 |
| 50-199 | 25% | 2,625円 |
| 200-399 | 22% | 2,730円 |
| 400+ | 18% | 2,870円 |

### iTalki/Verbling（15%固定）

| 表示料金 | 手数料後 |
|--------|---------|
| 2,500円 | 2,125円 |
| 3,000円 | 2,550円 |
| 3,500円 | 2,975円 |
| 4,000円 | 3,400円 |

## ステップ2：体験レッスンを考慮

Preplyでは、体験レッスンの受取額は0円です。月50レッスンを教え、5回が体験の場合：

- 有料レッスン45回 × 3,500円 = 157,500円（総額）
- 体験レッスン5回 × 3,500円 = 17,500円失う

無給時間を働いているため、実効料金が下がります。

**影響計算**：
```
失われた体験収入 = 体験レッスン × 時給
調整後総収入 = (有料レッスン × 料金) - 手数料
実際の料金 = 調整後総収入 ÷ 総レッスン数
```

**例**：
- 有料レッスン45回、25%手数料後：118,125円
- 無給体験レッスン5回：0円
- 総レッスン数：50
- 1レッスン時間あたりの実際の料金：2,363円（2,625円ではない）

---

## ステップ3：決済処理を含める

プラットフォームは手数料がかかるサービスを通じて支払います：

| サービス | 典型的な手数料 | 100,000円に対して |
|--------|-------------|----------------|
| PayPal | 2.5% | 2,500円 |
| Payoneer | 2% | 2,000円 |
| 銀行振込 | 1-3% | 1,000-3,000円 |

実際の料金にさらに2%の減少を追加。

---

## ステップ4：パッケージ割引を考慮

生徒が割引付きパッケージを購入し（その割引があなたの取り分から来る場合）：

- 10レッスンパッケージ、6%割引
- 1レッスンあたりの料金の94%を稼ぐ
- 3,500円の場合：約2,468円を得る（25%手数料後、割引価格に対して）

---

## ステップ5：管理時間を含める

教えることはレッスン時間だけではありません。生徒ごとの*合計*時間を計算：

| 活動 | 1レッスンあたりの時間 |
|-----|-----------------|
| 実際のレッスン | 60分 |
| レッスン準備 | 10-15分 |
| ノート/フォローアップ | 5-10分 |
| コミュニケーション | 5-10分 |
| **合計** | **80-95分** |

60分の教授で2,500円を稼ぎ、合計90分費やすなら、実際の時給は：

```
2,500円 ÷ 1.5時間 = 1,667円/時
```

## 完全な計算例

### シナリオ：Preply講師、3,500円/時、25%ティア

**月間活動**：
- 合計50レッスン
- 体験レッスン5回（無給）
- 有料レッスン45回
- 1レッスンあたり平均80分の作業

**計算**：

| ステップ | 金額 |
|--------|-----|
| 有料レッスン総額 | 157,500円 |
| 手数料（25%） | -39,375円 |
| 体験レッスン | -17,500円相当の作業 |
| Preplyからの純収入 | 118,125円 |
| 決済手数料（2%） | -2,363円 |
| **手取り** | **115,762円** |

**実際の時給計算**：

| 方法 | 計算 | 料金 |
|----|-----|-----|
| 1レッスン時間あたり | 115,762円 ÷ 50 | 2,315円/時 |
| 1作業時間あたり（80分/レッスン） | 115,762円 ÷ 66.7 | 1,736円/時 |

**表示料金：3,500円/時**
**実際の料金：1,736〜2,315円/時**

それは広告価格より34-50%少ない。

## 価格設定への意味

### 目標を達成するための料金設定

実際に（1レッスン時間あたり）3,000円/時を稼ぎたい場合：

| プラットフォーム | 手数料 | 設定すべき価格 |
|---------------|-------|-------------|
| Preply（新規） | 33% | 4,500円 |
| Preply（ベテラン） | 18% | 3,700円 |
| iTalki | 15% | 3,500円 |
| 直接 | 3% | 3,100円 |

### 直接予約の利点

直接3,000円/時 vs Preply3,500円/時（25%手数料）：

| 指標 | Preply 3,500円 | 直接 3,000円 |
|----|--------------|-----------|
| 生徒が支払う | 3,500円 | 3,000円 |
| あなたが受け取る | 2,625円 | 2,910円 |
| 生徒の節約 | - | 500円 |
| あなたの増加 | - | 285円 |

直接予約で双方がWin。

## アクションステップ

### 1. 現在の実際の料金を計算

今月の数字を使用：
- 総レッスン数（体験含む）
- 受け取った純額（すべての手数料後）
- 割り算：純額 ÷ レッスン数 = 1レッスン時間あたりの実際の料金

### 2. 目標を決める

1作業時間あたり（準備含む）実際にいくら稼ぎたいですか？

### 3. それに応じて価格設定

目標から逆算して、実際にそれを達成するプラットフォーム料金を設定。

### 4. ミックスを検討

直接予約ははるかに高い実際の料金。30%直接ミックスでも、全体の収入を大幅に改善。
