## はじめに

Preplyで教えている、または参加を検討している場合、手数料構造を理解することは家庭教師収入を計画する上で不可欠です。Preplyの料金は単純なパーセンテージではなく、経験レベルによって変動し、新規講師が驚くような料金も含まれています。

このガイドでは、2025年にPreplyが講師に請求する金額を、実際の例と計算で詳しく解説し、実際の手取り給与を理解できるようにします。

---

## Preplyの手数料構造

### 体験レッスン手数料：100%

多くの新規講師が驚く部分です：**Preplyは新しい生徒との最初のレッスンの100%を取ります。**

そうです—体験レッスンを3,000円に設定した場合、受け取るのは0円です。Preplyは生徒とのつながりの対価として全額を保持します。

Preplyの視点からの論理：生徒を獲得するために広告費を使った。体験レッスン手数料は顧客獲得コストをカバーする。

あなたの視点から：長期生徒を獲得する*機会*と引き換えに、本質的に無料レッスンを教えている。

### 継続的な手数料ティア

体験レッスン後、Preplyの手数料はプラットフォームでの教授時間によって変わります：

| 教授時間 | 手数料率 | 3,000円/時での手取り |
|---------|---------|-------------------|
| 0-19時間 | 33% | 2,010円 |
| 20-49時間 | 28% | 2,160円 |
| 50-199時間 | 25% | 2,250円 |
| 200-399時間 | 22% | 2,340円 |
| 400時間以上 | 18% | 2,460円 |

### 各ティアに到達するまでの時間

現実的なタイムラインを見てみましょう：

- **20時間**（28%に到達）：週5-7時間で約3-4週間
- **50時間**（25%に到達）：週6時間で約2ヶ月
- **200時間**（22%に到達）：週6時間で約8ヶ月
- **400時間**（18%に到達）：週6時間で約16ヶ月

ほとんどのパートタイム講師は最低手数料ティアに到達するまで**1-2年**かかります。

## 実際の収入計算

### 新規講師シナリオ（33%手数料）

**月間教授**：40レッスン、1時間2,500円

| 項目 | 金額 |
|-----|-----|
| 総予約 | 100,000円 |
| 体験レッスン（4回想定） | -10,000円（手取り0円） |
| 通常レッスン（36回 × 2,500円） | 90,000円（総額） |
| 手数料（90,000円の33%） | -29,700円 |
| **純収入** | **60,300円** |
| **実効時給** | **1,508円** |

広告時給は2,500円ですが、実際には1,508円しか稼いでいません。

### 経験豊富な講師シナリオ（18%手数料）

**月間教授**：40レッスン、1時間3,500円

| 項目 | 金額 |
|-----|-----|
| 総予約 | 140,000円 |
| 体験レッスン（2回想定） | -7,000円（手取り0円） |
| 通常レッスン（38回 × 3,500円） | 133,000円（総額） |
| 手数料（133,000円の18%） | -23,940円 |
| **純収入** | **109,060円** |
| **実効時給** | **2,727円** |

最良のティアでも、3,500円/時の講師の手取りは2,727円/時です。

## 手数料以外の隠れたコスト

### 支払い処理時間

PreplyはPayoneer、PayPal、またはSkrillを通じて講師に支払います。それぞれに手数料があります：

- **Payoneer**：2%の出金手数料 + 為替換算の可能性
- **PayPal**：場所により2-4%
- **Skrill**：1-2% + 換算手数料の可能性

### 「パッケージ割引」の影響

生徒はPreplyでレッスンパッケージを内蔵割引（通常5-10%）付きで購入できます。これらの割引はPreplyの手数料からではなく、*あなたの*収入から来ます。

例：生徒が5%割引の10レッスンパッケージを購入
- 通常料金：3,000円/時 × 10 = 30,000円
- 割引後：28,500円（生徒が支払う）
- Preplyの25%手数料：7,125円
- **あなたの収入：21,375円**（2,250円/時ではなく2,138円/時）

### キャンセルポリシー

生徒が4時間未満の通知でキャンセルした場合、100%の支払いを受けます。ただし、ポリシー内でキャンセルした場合は何ももらえません—そしてすでにその時間枠をブロックしていました。

## 他のプラットフォームとの比較

| プラットフォーム | 手数料 | 体験ポリシー | 最適な用途 |
|---------------|-------|------------|---------|
| Preply | 18-33% | 100%がプラットフォームへ | 高ボリューム、多言語 |
| iTalki | 15%固定 | 無料体験不要 | 低手数料、柔軟性 |
| Verbling | 15%固定 | 体験は様々 | ビデオ品質重視 |
| Cambly | 固定レート（約1,020円/時） | N/A | シンプル、マーケティング不要 |

Preplyの手数料は主要プラットフォームの中で最も高いですが、生徒獲得に多額の投資もしています。このトレードオフがあなたに合うかどうかは、他で生徒を引き付ける能力によります。

## Preplyの収入を最大化する戦略

### 1. 補償のために高めの価格設定

新規講師（33%手数料）として純2,500円/時を稼ぎたい場合：
```
目標時給 = 2,500円 ÷ (1 - 0.33) = 3,731円
```
実際に目標を達成するには3,800円/時に設定します。

### 2. 生徒の継続に集中

体験レッスンは本質的に無料マーケティングです。目標：体験生徒を長期のレギュラーに転換すること。

- 体験レッスンのために特に入念に準備
- 今後のレッスンの明確な計画で終了
- レッスンノートで迅速にフォローアップ

### 3. より多くの時間をより早く教える

手数料は各ティアで大幅に下がります。50時間以上に素早く到達できれば、将来のすべてのレッスンで8%節約できます。

より多くのレッスンを予約してティアを早く上げるために一時的に料金を下げることを検討し、25%以下の手数料に到達したら料金を上げます。

### 4. 長期戦略を考慮

Preplyは生徒ベースを構築するのに優れています。しかし、一貫したリピート生徒がいると、すべてのレッスンに永遠に18-33%のコストがかかります。

多くの講師がPreplyを戦略的に使用しています：
- Preplyを通じて新しい生徒を獲得
- 時間をかけて関係を構築
- 好む生徒には直接予約を利用可能に

## Preplyは価値があるか？

### Preplyが合理的な場合：
- 始めたばかりで迅速に生徒が必要
- 自分でマーケティングしたくない
- Preplyで需要の高い言語を教える
- プラットフォームのインフラ（スケジューリング、支払い、ビデオ）を重視

### 代替案を検討すべき場合：
- すでに生徒ベースがある
- 基本的なマーケティングに慣れている
- 長期的により多くの収入を保持したい
- Preplyで需要の低い言語を教える

---

## 結論

Preplyの手数料構造が意味すること：

- **新規講師**は通常レッスンの33% + 体験の100%を失う
- **経験豊富な講師**（400時間以上）でもまだ18% + 体験を失う
- **実際の時給**は広告時給より大幅に低い
- **パッケージ割引**はさらに収入を減らす

生徒獲得には、これらのコストは価値があるかもしれません。長期のリピート生徒には、もはや必要のないサービスにプレミアムを払っています。

これらの数字を理解することで、価格設定、プラットフォームミックス、代替予約チャネルの開発時期について情報に基づいた決定を下せます。
