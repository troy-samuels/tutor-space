## 요약 (TL;DR)

- **필수 도구**: 예약, 결제, 화상 통화, 캘린더 동기화
- **선택 도구**: CRM, 이메일 마케팅, 디지털 상품 판매
- **올인원 솔루션**이 10개 도구를 합친 것보다 효율적
- 월 2-5만 원 투자로 **주당 5-10시간 절약**
- 효율성 = 더 많은 레슨 시간 = 더 많은 수입

---

## 서론: 왜 테크 스택이 중요한가

독립 튜터로서 성공하려면 단순히 잘 가르치는 것만으로는 부족합니다. 효율적인 비즈니스 운영이 필요합니다.

**문제**: 대부분의 튜터는 10개 이상의 도구를 사용합니다:
- 예약: Calendly 또는 수동 이메일
- 결제: PayPal, 은행 송금, 카드
- 화상 통화: Zoom
- 학생 메모: Google Docs 또는 메모장
- 리마인더: 수동 메시지
- 마케팅: Instagram, 웹사이트
- 등등...

이 모든 것을 관리하는 데 **주당 5-10시간**을 낭비합니다.

## 2025년 필수 튜터 도구

### 1. 예약 시스템

**필요한 이유**: 이메일/메시지 왕복 없이 학생이 직접 예약

**기능 체크리스트**:
- [ ] 실시간 가용 시간 표시
- [ ] 시간대 자동 변환
- [ ] 자동 확인 이메일
- [ ] 재예약/취소 기능
- [ ] 캘린더 동기화

**옵션**:
| 도구 | 가격 | 튜터 특화 |
|------|------|-----------|
| TutorLingua | 무료/유료 | ✅ |
| Calendly | 무료/유료 | ❌ |
| Cal.com | 무료/유료 | ❌ |
| Acuity | 유료 | ❌ |

### 2. 결제 시스템

**필요한 이유**: 자동 결제, 영수증, 송장

**기능 체크리스트**:
- [ ] 카드 결제 수락
- [ ] 자동 영수증
- [ ] 구독/패키지 지원
- [ ] 환불 처리
- [ ] 다중 통화

**옵션**:
| 도구 | 수수료 | 통합 용이성 |
|------|--------|-------------|
| Stripe | ~3% | 높음 |
| PayPal | ~3-4% | 중간 |
| Square | ~3% | 중간 |

### 3. 화상 통화

**필요한 이유**: 안정적인 레슨 진행

**기능 체크리스트**:
- [ ] 안정적인 연결
- [ ] 화면 공유
- [ ] 화이트보드
- [ ] 녹화 (선택)
- [ ] 채팅

**옵션**:
| 도구 | 가격 | 특징 |
|------|------|------|
| Zoom | 무료/유료 | 가장 안정적 |
| Google Meet | 무료 | 간단함 |
| Microsoft Teams | 무료/유료 | 비즈니스용 |
| LiveKit | 유료 | 고품질 |

### 4. 캘린더 동기화

**필요한 이유**: 이중 예약 방지

**필수 기능**:
- Google Calendar 동기화
- Outlook Calendar 동기화
- 양방향 동기화

---

## 선택적이지만 유용한 도구

### 5. 학생 관리 (CRM)

학생이 10명을 넘으면 필요:
- 학생별 메모
- 진도 추적
- 레슨 이력
- 결제 이력

### 6. 자동 리마인더

노쇼를 50% 이상 줄입니다:
- 24시간 전 알림
- 1시간 전 알림
- 레슨 링크 포함

### 7. 이메일 마케팅

학생 유지 및 신규 획득:
- 뉴스레터
- 프로모션
- 리뷰 요청

### 8. 디지털 상품

추가 수입원:
- PDF 학습 자료
- 녹화된 강의
- 워크시트

## 테크 스택 구축 로드맵

### 1단계: 필수 요소 (1주차)

- [ ] 예약 시스템 설정
- [ ] 결제 연결
- [ ] 화상 통화 테스트

### 2단계: 효율성 (2-4주차)

- [ ] 캘린더 동기화
- [ ] 자동 리마인더 설정
- [ ] 학생 메모 시스템

### 3단계: 성장 (1-3개월)

- [ ] 전문적인 웹사이트/프로필
- [ ] 이메일 마케팅
- [ ] 디지털 상품 (선택)

## 시간 절약 계산

### 수동 vs 자동

| 작업 | 수동 | 자동 |
|------|------|------|
| 예약 조율 | 20분/건 | 0분 |
| 결제 확인 | 10분/건 | 0분 |
| 리마인더 발송 | 5분/건 | 0분 |
| 노쇼 재예약 | 15분/건 | 5분 |

**월 40건 예약 기준**:
- 수동: 33시간/월
- 자동: 3시간/월
- **절약: 30시간/월**

30시간 × 50,000원/시간 = **월 150만 원 가치**

## 결론: 투자 대비 수익

올바른 테크 스택은:
- 주당 5-10시간 절약

월 5만 원 투자로 월 100만 원+ 가치 창출이 가능합니다.
