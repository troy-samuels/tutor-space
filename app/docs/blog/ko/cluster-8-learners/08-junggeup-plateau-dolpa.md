## 요약 (TL;DR)

- **중급 정체기**는 B1-B2 단계에서 진전이 멈춘 느낌을 받는 현상
- 원인: 일상 대화 가능 → 도전 부족, 쉬운 것만 하게 됨
- 극복법: **불편한 상황** 의도적으로 찾기, 약점 집중 공략
- 정체기는 **실패가 아닌 정상** — 대부분의 학습자가 경험
- 튜터와 함께하면 **돌파 속도가 2배**

---

## 서론: "왜 더 이상 안 늘어?"

처음 6개월은 매주 발전이 느껴졌습니다. 새 단어, 새 문법, 새로운 대화 능력.

그런데 어느 순간부터... 멈춘 것 같습니다.

- 기본 대화는 되는데
- 뉴스는 아직 어렵고
- 복잡한 생각은 표현 못하고
- 실력이 그대로인 것 같고

**축하합니다. 중급 정체기에 도착했습니다.**

## 중급 정체기란?

### 정의

중급 정체기(Intermediate Plateau)는 언어 학습에서 특정 단계(보통 B1-B2)에서 진전이 급격히 느려지거나 멈춘 것처럼 느껴지는 현상입니다.

### 증상

- ✅ 일상 대화는 가능
- ✅ 기본적인 것은 이해
- ❌ 복잡한 주제 논의 어려움
- ❌ 뉴스/영화 완전 이해 어려움
- ❌ 미묘한 뉘앙스 표현 어려움
- ❌ 실력 향상이 안 느껴짐

### 언제 발생하나?

| 레벨 | 증상 |
|------|------|
| A1-A2 | 빠른 진전, 매주 새로운 것 |
| B1 | 정체기 시작, 기본은 되는데... |
| B2 | 정체기 지속, 고급으로 넘어가기 어려움 |
| C1+ | 정체기 극복, 미세 조정 단계 |

---

## 왜 발생하나? (5가지 원인)

### 원인 1: 도전 회피

**현상**: "편한 것"만 하게 됨

기본 대화가 되면:
- 쉬운 콘텐츠만 선택
- 같은 주제만 이야기
- 어려운 상황 피함

**결과**: 더 이상 성장 압력 없음

### 원인 2: 같은 실수 반복

**현상**: 잘못된 습관이 굳어짐

오랜 시간 같은 실수를:
- 교정받지 않고
- 인식하지 못하고
- 계속 사용

**결과**: "화석화" — 실수가 영구적으로 고착

### 원인 3: 수동적 학습 의존

**현상**: 듣기/읽기만, 말하기/쓰기 부족

중급이 되면:
- 넷플릭스로 "공부"
- 팟캐스트로 "학습"
- 능동적 출력 부족

**결과**: 입력은 늘지만 출력은 정체

### 원인 4: 어휘의 벽

**현상**: 일상 어휘는 알지만 고급 어휘 부족

일상 대화에 필요한 단어: ~2,000개
유창한 대화에 필요한 단어: ~5,000-10,000개

**결과**: 표현력 한계

### 원인 5: 문법의 미세 부분

**현상**: 기본 문법은 알지만 미묘한 차이 모름

- 접속법 vs 직설법
- 시제의 미세한 차이
- 격식/비격식 구분
- 관용적 표현

**결과**: "맞지만 부자연스러운" 언어

## 극복 전략 (5가지)

### 전략 1: 의도적 불편함

**원칙**: 편안한 영역 밖으로 나가기

**실천법**:
- 어려운 주제로 대화 요청
- 뉴스/다큐멘터리 시청
- 원서 읽기 (70% 이해 수준)
- 네이티브 팟캐스트 듣기

**목표**: 매일 "살짝 어려운" 경험

### 전략 2: 출력 중심 학습

**원칙**: 말하기/쓰기 비율 높이기

**실천법**:
- 매일 일기 쓰기 (목표 언어로)
- 혼잣말 연습 (생각을 소리내어)
- 언어 교환 파트너
- 튜터와 토론 세션

**목표**: 학습 시간의 50% 이상을 출력에

### 전략 3: 약점 집중 공략

**원칙**: 피하던 것을 정면 돌파

**실천법**:
1. 자주 하는 실수 5개 리스트
2. 각 실수에 대한 규칙 학습
3. 해당 구조를 사용한 문장 20개 작성
4. 튜터에게 교정 받기
5. 반복

**목표**: 매주 하나의 약점 극복

### 전략 4: 고급 입력

**원칙**: 입력 수준 올리기

**실천법**:
| 현재 | 목표 |
|------|------|
| 학습자용 콘텐츠 | 네이티브용 콘텐츠 |
| 자막 있음 | 자막 없음 |
| 천천히 말하는 것 | 자연스러운 속도 |
| 익숙한 주제 | 새로운 주제 |

### 전략 5: 전문가 피드백

**원칙**: 혼자 하면 보이지 않는 것이 있음

**왜 튜터가 필요한가**:
- 인식하지 못하는 실수 발견
- 맞춤형 개선 계획
- 동기부여 유지
- 객관적 진전 측정

**권장**: 주 1-2회 집중 세션

## 정체기는 얼마나 지속되나?

### 일반적인 타임라인

| 요인 | 짧은 정체기 | 긴 정체기 |
|------|------------|----------|
| 학습 방법 | 다양함, 능동적 | 단조로움, 수동적 |
| 피드백 | 정기적 | 없음 |
| 도전 수준 | 점진적 증가 | 일정 |
| 기간 | 2-4개월 | 6개월-수년 |

### 중요한 진실

- 정체기는 **정상**입니다
- 진전이 **눈에 안 보일 뿐** 일어나고 있을 수 있음
- 많은 사람이 이 단계에서 **포기**
- 극복하면 **급속 성장**이 가능

---

## 진전 측정하기

### 객관적 지표

1. **녹음 비교**: 3개월 전 vs 현재 말하기
2. **쓰기 샘플**: 이전 글과 비교
3. **이해도 테스트**: 같은 콘텐츠 재시청
4. **어휘 테스트**: 알고 있는 단어 수

### 주관적 지표

1. 대화에서 **편안함** 증가
2. **검색 없이** 표현 가능한 것 증가
3. **이해 속도** 향상
4. **자신감** 증가

---

## 자주 묻는 질문

### "정체기인지 어떻게 알아요?"

- 3개월 이상 진전이 느껴지지 않음
- 기본은 되는데 그 이상 어려움
- 학습 동기가 떨어짐
- 같은 실수 반복

### "앱으로 극복 가능한가요?"

부분적으로만. 앱은:
- 어휘 확장에 유용
- 문법 복습에 유용
- 하지만 **말하기 연습과 피드백**은 불가

### "얼마나 걸리나요?"

의도적 노력으로:
- 경미한 정체기: 1-2개월
- 심각한 정체기: 3-6개월
- 노력 없이: 무기한

### "포기해야 하나요?"

절대 아닙니다! 정체기는:
- 정상적인 학습 과정
- 극복 가능함
- 극복 후 빠른 성장

---

## 결론: 정체기는 통과점

중급 정체기는:
- **실패**가 아님
- **영구적**이지 않음
- **극복 가능**함

필요한 것:
1. **인식**: 무슨 일인지 이해
2. **전략**: 방법 변화
3. **지속**: 포기하지 않기
4. **도움**: 전문가 피드백

정체기 너머에는 유창함이 기다리고 있습니다.
