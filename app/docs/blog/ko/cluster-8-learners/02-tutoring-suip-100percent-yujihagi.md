## 요약 (TL;DR)

- 마켓플레이스는 **15-33%** 수수료 부과 — 연간 수백만 원
- **반복 학생**은 직접 예약으로 전환 가능
- 직접 예약에 필요한 것: 예약 페이지 + 결제 시스템
- **하이브리드 전략**: 마켓플레이스로 발굴, 직접 예약으로 유지
- 결제 수수료 ~3%만 지불하고 나머지 100% 유지

---

## 서론: 수수료의 현실

Preply에서 월 200만 원을 번다면, 실제로 **최대 66만 원**이 플랫폼에 갑니다. 이는 연간 **792만 원**입니다.

좋은 소식? 이 학생들 중 상당수는 직접 예약을 통해 계속 레슨을 받을 수 있고, 그러면 수입의 거의 전부를 유지할 수 있습니다.

---

## 왜 마켓플레이스에서 완전히 떠나면 안 되는가

먼저 명확히 해야 할 점: **마켓플레이스를 완전히 버리라는 게 아닙니다**.

마켓플레이스의 가치:
- 새로운 학생 발굴
- 초기 신뢰 구축
- 리뷰 축적

마켓플레이스의 문제:
- **반복 학생**에게도 계속 수수료 지불
- 이미 신뢰 관계가 형성된 학생에게서도 15-33% 가져감

**핵심 전략**: 마켓플레이스는 학생 발굴용으로 사용하고, 반복 학생은 직접 예약으로 전환

---

## 직접 예약에 필요한 것

### 1. 예약 시스템

학생이 직접 시간을 선택하고 예약할 수 있는 시스템이 필요합니다.

**옵션**:
- TutorLingua (튜터 전용)
- Calendly
- Cal.com
- 직접 구축 (복잡)

### 2. 결제 시스템

결제를 받을 방법이 필요합니다.

**옵션**:
- Stripe (카드 결제, ~3%)
- PayPal (~3-4%)
- 은행 송금 (수수료 없음, 수동 확인 필요)

### 3. 리마인더 시스템

노쇼를 줄이기 위한 자동 리마인더:
- 24시간 전 알림
- 1시간 전 알림
- 레슨 링크 포함

---

## 학생에게 직접 예약을 제안하는 방법

### 언제 제안할까

**좋은 타이밍**:
- 4-8회 레슨 후 (관계 형성 후)
- 장기 계획을 논의할 때
- 학생이 계속하고 싶다고 할 때

**피해야 할 타이밍**:
- 첫 레슨 중
- 플랫폼 메시지 내에서
- 학생이 아직 결정하지 않았을 때

### 무엇을 말할까

**간단하고 직접적으로**:
> "레슨 즐기고 계신 것 같아서 기뻐요! 장기적으로 계속하신다면, [플랫폼] 외부에서 직접 예약하는 옵션이 있어요. 제 예약 페이지로 바로 예약하실 수 있어요."

**비용 절감을 언급하고 싶다면**:
> "플랫폼이 꽤 높은 수수료를 가져가는 걸 알고 계실 수도 있어요. 직접 예약하시면 더 좋은 가격에 레슨을 드릴 수 있어요 — 우리 둘 다 이득이에요."

---

## 가격 설정 전략

직접 예약 시 가격을 어떻게 설정할까요?

### 옵션 1: 동일한 가격

- 학생은 같은 금액 지불
- 튜터 수입 증가
- 가장 간단한 전환

### 옵션 2: 약간 할인

- 학생에게 5-10% 할인 제공
- 튜터는 여전히 더 많이 벌음
- 윈-윈 상황

### 예시

마켓플레이스 가격: 50,000원/시간 (33% 수수료 후 33,500원 수령)

**직접 예약 옵션**:
- 50,000원 → 48,500원 수령 (15,000원 증가)
- 45,000원 → 43,650원 수령 (학생 할인, 여전히 10,150원 증가)

## 일반적인 우려 사항

### "플랫폼에서 허용되나요?"

플랫폼은 일반적으로 **플랫폼 내** 연락처 공유를 금지합니다. 하지만:
- 플랫폼 외부에서의 연락은 통제 불가
- 대화가 플랫폼 밖에서 이루어짐
- 당신의 돈, 당신의 선택

### "학생이 떠날까봐 걱정됩니다"

직접 예약은 관계를 **강화**합니다:
- 더 개인화된 서비스
- 더 유연한 정책
- 더 직접적인 소통

### "기술적으로 어렵지 않나요?"

현대 도구들은 설정을 매우 쉽게 만들어줍니다:
- 대부분 30분 내 설정 가능
- 기술 지식 거의 불필요
- 많은 무료 옵션 존재

---

## 성공적인 전환 타임라인

### 1-3개월: 설정

- [ ] 예약 페이지 생성
- [ ] 결제 시스템 연결
- [ ] 자동 리마인더 설정

### 4-6개월: 전환

- [ ] 반복 학생에게 직접 예약 제안
- [ ] 새 학생은 마켓플레이스에서 계속 받음
- [ ] 점진적으로 직접 예약 비율 증가

### 6-12개월: 최적화

- [ ] 직접 예약 학생 50% 이상 목표
- [ ] 프로세스 개선
- [ ] 추천 시스템 구축

## 결론

마켓플레이스 수수료는 피할 수 없는 것이 아닙니다. 전략적 접근으로:

1. **마켓플레이스**는 학생 발굴에 사용
2. **직접 예약**은 반복 학생 유지에 사용
3. 수입의 거의 **100%**를 유지

연간 수백만 원을 절약하면서도 학생들과 더 좋은 관계를 구축할 수 있습니다.
