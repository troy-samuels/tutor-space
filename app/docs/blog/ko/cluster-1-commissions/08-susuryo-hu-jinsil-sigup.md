## 서론

Preply에서 시간당 $35를 청구합니다. 하지만 실제로 *얼마를 버나요*?

수수료, 체험 레슨, 결제 처리, 기타 비용을 빼면, 실제 시급은 생각보다 훨씬 낮을 수 있습니다. 이 숫자를 이해하는 것은 가격 결정과 재무 계획에 필수입니다.

실제 수입을 계산해 봅시다.

---

## 실제 시급 공식

실제 시급은 표시 가격이 아닙니다. 이것입니다:

```
실제 시급 = (총수입 - 모든 비용) ÷ 총 근무 시간
```

"총 근무 시간"에는 레슨 시간과 관리 시간(스케줄링, 준비, 커뮤니케이션)이 포함됩니다.

---

## 1단계: 플랫폼 수수료 계산

### Preply 등급별 수수료

| 강의 시간 | 수수료 | $35 레슨 시 수령액 |
|----------|-------|------------------|
| 0-19 | 33% | $23.45 |
| 20-49 | 28% | $25.20 |
| 50-199 | 25% | $26.25 |
| 200-399 | 22% | $27.30 |
| 400+ | 18% | $28.70 |

### iTalki/Verbling (15% 고정)

| 표시 요금 | 수수료 후 |
|---------|---------|
| $25 | $21.25 |
| $30 | $25.50 |
| $35 | $29.75 |
| $40 | $34.00 |

---

## 2단계: 체험 레슨 반영

Preply에서 체험 레슨의 수령액은 $0입니다. 월 50레슨 중 5회가 체험이면:

- 유료 레슨 45회 × $35 = $1,575 총액
- 체험 레슨 5회 × $35 = $175 손실

무급 시간을 일하기 때문에 실효 요금이 떨어집니다.

**영향 계산**:
```
손실 체험 수입 = 체험 레슨 × 시급
조정된 총수입 = (유료 레슨 × 요금) - 수수료
실제 요금 = 조정된 총수입 ÷ 총 레슨 수
```

**예시**:
- 25% 수수료 후 유료 레슨 45회: $1,181.25
- 무급 체험 레슨 5회: $0
- 총 레슨 수: 50
- 레슨 시간당 실제 요금: $23.63 ($26.25가 아님)

---

## 3단계: 결제 처리 포함

플랫폼은 수수료가 있는 서비스를 통해 지급합니다:

| 서비스 | 일반적인 수수료 | $1,000에 대해 |
|--------|---------------|-------------|
| PayPal | 2.5% | $25 |
| Payoneer | 2% | $20 |
| 은행 송금 | 1-3% | $10-30 |

실제 요금에 추가로 2% 감소를 더하세요.

---

## 4단계: 패키지 할인 고려

학생이 할인 패키지를 구매하면 (할인이 당신 몫에서 나오면):

- 6% 할인 10레슨 패키지
- 레슨당 요금의 94%를 법
- $35의 경우: ~$24.68 (25% 수수료 후, 할인가에 대해)

---

## 5단계: 관리 시간 포함

강의는 레슨 시간만이 아닙니다. 학생당 *총* 시간을 계산하세요:

| 활동 | 레슨당 시간 |
|------|-----------|
| 실제 레슨 | 60분 |
| 레슨 준비 | 10-15분 |
| 노트/후속 조치 | 5-10분 |
| 커뮤니케이션 | 5-10분 |
| **총합** | **80-95분** |

60분 강의로 $25를 벌고 총 90분을 쓴다면, 실제 시급은:

```
$25 ÷ 1.5시간 = $16.67/시간
```

## 완전한 계산 예시

### 시나리오: Preply 강사, $35/시간, 25% 등급

**월간 활동**:
- 총 50레슨
- 체험 레슨 5회 (무급)
- 유료 레슨 45회
- 레슨당 평균 80분 작업

**계산**:

| 단계 | 금액 |
|------|------|
| 유료 레슨 총액 | $1,575 |
| 수수료 (25%) | -$393.75 |
| 체험 레슨 | -$175 상당의 작업 |
| Preply에서 순수입 | $1,181.25 |
| 결제 수수료 (2%) | -$23.63 |
| **실수령액** | **$1,157.62** |

**실제 시급 계산**:

| 방법 | 계산 | 요금 |
|------|------|------|
| 레슨 시간당 | $1,157.62 ÷ 50 | $23.15/시간 |
| 근무 시간당 (80분/레슨) | $1,157.62 ÷ 66.7 | $17.36/시간 |

**표시 요금: $35/시간**
**실제 요금: $17.36-$23.15/시간**

광고 가격보다 34-50% 적습니다.

## 가격 책정에 대한 의미

### 목표를 달성하는 요금 설정

실제로 (레슨 시간당) $30/시간을 벌고 싶다면:

| 플랫폼 | 수수료 | 설정할 가격 |
|--------|-------|----------|
| Preply (신규) | 33% | $45 |
| Preply (베테랑) | 18% | $37 |
| iTalki | 15% | $35 |
| 직접 | 3% | $31 |

### 직접 예약의 이점

직접 $30/시간 vs Preply $35/시간 (25% 수수료):

| 지표 | Preply $35 | 직접 $30 |
|------|-----------|---------|
| 학생 지불액 | $35 | $30 |
| 당신 수령액 | $26.25 | $29.10 |
| 학생 절약 | - | $5 |
| 당신 추가 | - | $2.85 |

직접 예약으로 양측 모두 윈.

## 실행 단계

### 1. 현재 실제 요금 계산

이번 달 숫자로:
- 총 레슨 수 (체험 포함)
- 총 순수령액 (모든 수수료 후)
- 나누기: 순수령 ÷ 레슨 수 = 레슨 시간당 실제 요금

### 2. 목표 결정

실제로 시간당 (준비 포함) 얼마를 벌고 싶은가?

### 3. 그에 맞게 가격 책정

목표에서 역산하여 실제로 달성하는 플랫폼 요금을 설정.

### 4. 조합 고려

직접 예약은 실제 요금이 훨씬 높습니다. 30%만 직접이어도 전체 수입이 상당히 개선됩니다.
