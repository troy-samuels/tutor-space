## 서론

6개월 동안 Preply에서 Maria를 가르쳤습니다. 그녀는 매주 예약하고, 레슨을 좋아하며, 친구 두 명을 소개해 주었습니다. 어디 가지 않을 것입니다.

그런데도 매 레슨마다 몇 달 전에 "가치 제공"이 끝난 "서비스"에 대해 25% 수수료를 지불하고 있습니다. Maria는 Preply를 통해 당신을 찾았지만, 이제 그녀는 *당신의* 학생입니다—그리고 당신은 이미 구축한 관계에 대해 본질적으로 임대료를 내고 있는 것입니다.

문제는 기존 학생에게 직접 예약을 제공할 *것인지 여부*가 아닙니다. *어떻게* 윤리적이고 효과적으로 할 것인지입니다.

---

## 직접 예약의 윤리

이것을 직접적으로 다루어 봅시다: 이용약관을 위반하면서 학생을 플랫폼에서 데려가는 것은 잘못된 것입니다. 생태계를 손상시키고 밴당할 수 있습니다.

하지만 뉘앙스 있는 현실은 이렇습니다:

### 괜찮지 않은 것
- 플랫폼을 사용하면서 학생에게 플랫폼 외부 예약을 권유하는 것
- 플랫폼 메시징을 통해 연락처를 공유하는 것
- 학생에게 플랫폼을 떠나라고 압박하는 것
- 학생에게 플랫폼을 비난하는 것

### 괜찮은 것
- 자신의 전문적인 프레즌스(웹사이트, 소셜 미디어)를 갖는 것
- 학생이 독립적인 검색을 통해 당신을 찾게 하는 것
- 학생이 스스로 정보에 입각한 선택을 하도록 하는 것
- 학생이 선택한 곳에서 서비스를 제공하는 것

핵심적인 구분: **학생을 데려가는 것이 아니라—자신을 발견 가능하게 만들고, 학생이 어디서 예약할지 선택하는 것입니다.**

---

## 학생이 직접 예약을 원하는 이유

이것은 당신의 수입만의 문제가 아닙니다. 학생들도 진정으로 혜택을 받습니다:

### 1. 더 낮은 가격
수수료에서 25%를 절약하면 학생에게 절약을 전달할 수 있습니다. $30 Preply 레슨이 $25 직접 레슨이 됩니다—같은 강사, 같은 품질, 더 낮은 가격.

### 2. 더 많은 유연성
플랫폼 규칙은 스케줄링, 취소 정책, 레슨 형식을 제한합니다. 직접 예약은 경험을 맞춤화할 수 있습니다.

### 3. 직접적인 관계
일부 학생은 플랫폼 메시징보다 이메일이나 WhatsApp으로 직접 소통하는 것을 선호합니다.

### 4. 간소화된 예약
학생이 당신의 가능 시간을 알게 되면, 플랫폼 인터페이스를 탐색하는 것보다 직접 예약을 선호할 수 있습니다.

---

## 먼저 필요한 인프라

직접 예약을 받기 전에, 플랫폼이 제공하는 것과 동일한 기능이 필요합니다:

### 필수 구성 요소

**1. 온라인 스케줄링**
학생이 가능 시간을 보고 즉시 예약할 수 있어야 합니다—이메일 왕복 없이.

**2. 결제 처리**
카드, PayPal 또는 지역 결제 수단을 받아야 합니다. 은행 송금을 따로 요청하는 것은 지속 가능하지 않습니다.

**3. 자동화된 커뮤니케이션**
예약 확인, 알림, 영상 링크가 자동으로 발송되어야 합니다.

**4. 전문적인 프레즌스**
간단한 웹사이트나 랜딩 페이지가 플랫폼 외부에서 신뢰성을 확보합니다.

별도의 도구(Calendly + Stripe + Mailchimp)로 구축하거나 모든 것을 결합한 TutorLingua 같은 올인원 솔루션을 사용할 수 있습니다.

## 수동적 발견 방식

직접 예약에 대한 가장 윤리적인 접근법은 **수동적 발견**입니다—적극적으로 권유하지 않고 발견될 수 있게 만드는 것.

### 1단계: 전문적인 프레즌스 만들기

다음이 포함된 간단한 웹사이트나 랜딩 페이지를 만드세요:
- 자기소개와 교수 철학
- 서비스와 가격 (직접 예약 할인 포함)
- 명확한 예약 링크
- 전문적인 사진
- 추천글 (동의받은 경우)

### 2단계: 이름 검색 최적화

학생이 당신의 이름을 Google에 검색할 때, 웹사이트가 나타나야 합니다. 기본 SEO:
- 페이지 제목에 본명 사용
- 메타 설명에 "언어 강사" 포함
- 교습 언어와 전문 분야 추가
- 관련이 있다면 Google 비즈니스 등록

### 3단계: 소셜 미디어 구축

학생이 사용하는 플랫폼에 전문 프로필 만들기:
- Instagram (언어 강사에게 좋음)
- LinkedIn (비즈니스 영어 학생용)
- Facebook 페이지

모든 바이오에 예약 링크를 포함하세요.

### 4단계: 학생이 찾게 하기

플랫폼 외부에서 당신을 찾고 싶은 학생은 검색할 것입니다. 그렇게 할 때, 직접 예약 옵션을 발견하고 스스로 선택할 수 있습니다.

---

## 자연스러운 전환 접근법

일부 학생은 어떤 유도 없이도 자연스럽게 전환합니다:

### 이름 검색
몇 달 동안 함께한 학생은 종종 호기심에서 당신의 이름을 Google에 검색합니다. 더 낮은 직접 요금이 있는 웹사이트를 찾으면, 연락할 수 있습니다.

### 소셜 연결
학생이 Instagram이나 LinkedIn에서 당신을 팔로우하면, 바이오에서 예약 링크를 봅니다. 준비가 되면 옵션이 거기 있습니다.

### 소개 요청
학생이 친구를 소개하고 싶을 때, 공유할 링크를 물을 수 있습니다. 직접 예약 페이지는 소개에 완벽합니다—그리고 새 학생은 처음부터 직접으로 시작합니다.

### 구전
행복한 학생은 친구에게 당신을 언급합니다. 그 친구들은 당신의 이름을 검색하고 웹사이트를 찾습니다—원래 플랫폼을 전혀 거치지 않고요.

## 직접 레슨 가격 책정

목표는 윈윈입니다: 학생은 돈을 절약하고, 당신은 더 많이 버는 것.

### 윈윈 공식

```
플랫폼 가격: $35/시간
플랫폼 수수료: 25%
당신 몫: $26.25/시간

직접 가격: $30/시간
당신 몫: ~$29/시간 (결제 처리 ~3% 후)

학생 절약: $5/시간
당신 추가 수입: $2.75/시간
```

### 가격 전략

**옵션 A: 단순 할인**
직접 가격 = 플랫폼 가격 - 10-15%

**옵션 B: 가치 추가**
플랫폼과 같은 가격, 하지만 부가 혜택 포함 (레슨 자료, 숙제 검토, WhatsApp 접근)

**옵션 C: 패키지 인센티브**
단일 레슨은 플랫폼 요금, 패키지(10레슨 이상)는 직접 예약 할인

## 전환 대화 처리하기

학생이 직접 예약에 대해 문의할 경우 (웹사이트를 찾았거나, 친구가 소개했거나 등), 전문적으로 처리하는 방법:

### 해야 할 말:
- "네, 제 웹사이트를 통해 직접 예약을 받습니다"
- "가격은 [X]이고, 플랫폼보다 조금 낮습니다"
- "편한 곳에서 예약하시면 됩니다"
- "어디서 예약하든 같은 품질을 제공하겠습니다"

### 하지 말아야 할 말:
- "Preply가 너무 많이 가져가니까 떠나셔야 해요"
- "전환하시면 더 큰 할인을 드릴게요"
- "플랫폼이 우리 둘 다 속이고 있어요"
- 플랫폼에 대해 부정적인 말

전문적으로 유지하고 학생이 스스로 정보에 입각한 결정을 내리게 하세요.

## 조합 관리하기

대부분의 강사는 플랫폼과 직접 학생의 조합으로 끝납니다. 관리 방법:

### 동등한 서비스
예약 출처에 관계없이 동일한 레슨 품질을 제공합니다. 평판은 일관성에 달려 있습니다.

### 별도 추적
회계 목적으로 어느 학생이 어느 채널로 예약하는지 명확한 기록을 유지합니다.

### 플랫폼 규칙 준수
플랫폼 학생에 대해서는 계속 플랫폼 약관을 따릅니다. 플랫폼 메시징으로 직접 예약을 홍보하지 마세요.

### 점진적 전환
시간이 지나면, 더 많은 새 학생이 직접 옵니다 (소개, 소셜 미디어, SEO를 통해). 플랫폼 비율이 자연스럽게 감소합니다.

## 숫자로 보기: 장기적 영향

2년 후 강사를 모델링해 봅시다:

### 1년차 (전부 플랫폼)
- 50명 학생, 전부 Preply 경유
- 평균 25% 수수료
- 1,000 레슨 × $30 = $30,000 총수입
- 수수료: $7,500
- **순수입: $22,500**

### 2년차 (혼합 전략)
- 30명 플랫폼 학생 (신규 + 플랫폼 선호자)
- 35명 직접 학생 (전환 + 소개)
- 플랫폼 레슨: 400 × $30 = $12,000 총수입 → $9,000 순수입
- 직접 레슨: 700 × $28 = $19,600 총수입 → $19,000 순수입 (처리 후)
- **총 순수입: $28,000**

**개선: 연간 $5,500** (24% 증가)

## 플랫폼 충성도는 어떨까?

일부 강사는 직접 예약에 대해 죄책감을 느낍니다. 이 관점을 고려해 보세요:

플랫폼은 진정한 가치를 제공했습니다: 혼자서는 닿을 수 없었던 학생과 연결해 주었습니다. 초기 레슨에 대한 수수료로 그 서비스에 대해 지불했습니다.

하지만 장기 학생과의 관계는 품질 높은 강의, 일관성, 관심을 통해 *당신이* 구축한 것입니다. 플랫폼이 소개를 촉진했지만—관계를 만들지는 않았습니다.

몇 달 또는 몇 년 전에 제공된 서비스에 대해 영원히 수수료를 지불하는 것은 충성이 아닙니다—비효율입니다.

## 시작하기

**1주차**: 직접 예약 인프라 설정 (스케줄링, 결제, 기본 랜딩 페이지)

**2주차**: 예약 링크가 있는 소셜 미디어 프로필 생성 또는 최적화

**3주차**: 이름이 검색 가능하고 웹사이트가 결과에 나타나도록 확인

**4주차 이후**: 훌륭한 강의를 계속하세요. 학생이 선호하는 예약 방법을 발견하고 선택하게 하세요.

전환은 시간이 지나면서 자연스럽게 일어납니다. 압박 없이, 권유 없이—그냥 전문적인 가용성입니다.
