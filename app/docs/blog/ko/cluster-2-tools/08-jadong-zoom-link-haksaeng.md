# 학생에게 자동으로 Zoom 링크 보내는 방법: 완전 가이드

"선생님, Zoom 링크가 어디 있어요?"

이 메시지를 받는 것에 지쳤나요? 수업 시작 5분 전에 학생이 링크를 찾지 못해 허둥대는 상황...

이 문제는 완전히 예방 가능합니다. 자동화를 설정하면 예약이 확정되는 순간 Zoom 링크가 자동으로 생성되어 학생에게 발송됩니다.

이 가이드에서 가장 인기 있는 도구별로 자동 Zoom 링크를 설정하는 방법을 단계별로 알려드립니다.

## 왜 자동 Zoom 링크가 중요한가

### 수동 링크의 문제

수동으로 Zoom 링크를 보내면:

1. **시간 낭비**: 예약당 2-3분 × 주 20개 예약 = 주 1시간
2. **잊어버림**: 바쁘면 링크 보내는 것을 잊음
3. **혼란**: 학생이 이전 링크를 사용하려 함
4. **지연**: 수업 시작 시간에 링크 찾느라 허비

### 자동화의 이점

- **0분 투자**: 한 번 설정하면 영원히 작동
- **일관성**: 모든 예약에 동일한 경험
- **전문성**: 즉시 확인 이메일 발송
- **노쇼 감소**: 링크가 항상 포함되어 핑계 제거

## 방법 1: Calendly + Zoom

가장 간단한 설정. 5분이면 완료됩니다.

### 사전 요구사항
- Calendly 계정 (무료 티어 가능)
- Zoom 계정 (무료 티어 가능)

### 설정 단계

**1단계: Calendly에서 Zoom 연결**
1. Calendly 로그인
2. Account → Integrations 클릭
3. "Zoom" 찾아서 "Connect" 클릭
4. Zoom 로그인하여 권한 승인

**2단계: 이벤트 유형에 Zoom 적용**
1. Event Types로 이동
2. 편집할 이벤트 클릭
3. "Location" 섹션에서 "Zoom" 선택
4. 저장

**3단계: 테스트**
1. 자신의 예약 페이지로 이동
2. 테스트 예약 진행
3. 확인 이메일에 Zoom 링크가 있는지 확인

### 결과
- 학생이 예약하면 → 고유한 Zoom 미팅이 자동 생성
- 확인 이메일에 Zoom 링크 포함
- 리마인더 이메일에도 Zoom 링크 포함
- 캘린더 이벤트에 자동 추가

## 방법 2: Google Calendar + Zoom

이미 Google Calendar를 사용한다면 직접 통합할 수 있습니다.

### 사전 요구사항
- Google Workspace 계정 (또는 개인 Google 계정)
- Zoom 계정

### 설정 단계

**1단계: Google Calendar에 Zoom 추가**
1. Google Calendar 열기
2. 설정 ⚙️ → "Add-ons" 클릭
3. "Zoom for Google Workspace" 검색 및 설치
4. Zoom 계정으로 로그인

**2단계: 이벤트에 Zoom 추가**
1. 새 이벤트 생성
2. "Add Google Meet video conferencing" 옆의 드롭다운 클릭
3. "Zoom Meeting" 선택
4. 저장

### 제한사항
- 각 이벤트에 수동으로 Zoom 선택해야 함
- 예약 시스템과 별도 (Calendly만큼 자동화 안 됨)
- 학생에게 캘린더 초대 수동 발송 필요

## 방법 3: Acuity Scheduling + Zoom

Acuity는 Calendly보다 더 많은 옵션을 제공합니다.

### 설정 단계

**1단계: Zoom 연결**
1. Acuity 로그인
2. Integrations 메뉴
3. "Zoom" 찾아서 연결

**2단계: 예약 유형에 적용**
1. Appointment Types로 이동
2. 각 유형 편집
3. "Location" 에서 Zoom 선택
4. 저장

### 추가 옵션
- 특정 예약 유형에만 Zoom 적용
- 대안 화상 도구 설정 (Google Meet, Microsoft Teams)
- 커스텀 Zoom 설정 (대기실, 비밀번호 등)

## 방법 4: TutorLingua + Zoom

TutorLingua는 튜터를 위해 설계되어 설정이 가장 간단합니다.

### 설정 단계

**1단계: 화상 설정 연결**
1. TutorLingua 대시보드 → Settings
2. "Video" 탭 클릭
3. "Connect Zoom" 클릭
4. Zoom 로그인하여 권한 승인

**2단계: 완료!**
- 모든 새 예약에 자동으로 Zoom 링크 생성
- 확인 이메일에 포함
- 리마인더에 포함
- 학생 포털에서 바로 접근 가능

### TutorLingua의 추가 기능
- 모든 수업 녹화 (Studio 플랜)
- 수업 노트와 Zoom 링크 연결
- 학생별 수업 이력에서 링크 확인

## 대안: 고정 Personal Meeting ID

매번 새 미팅을 생성하는 대신, 고정 링크를 사용할 수도 있습니다.

### 설정 방법
1. Zoom 로그인 → Profile
2. "Personal Meeting ID" 확인
3. 이 링크를 모든 예약 확인에 사용

### 장점
- 설정 불필요
- 항상 같은 링크
- 학생이 기억하기 쉬움

### 단점
- 보안 취약 (누구나 들어올 수 있음)
- 예약 시간 중복 시 충돌
- 미팅 기록이 섞임

### 권장사항
- **1:1 단골 학생**: Personal Meeting ID 괜찮음
- **새 학생/그룹**: 자동 생성된 고유 링크 권장

## Zoom 대안: 다른 화상 도구

Zoom 외에도 옵션이 있습니다:

### Google Meet
- 장점: Google 계정만 있으면 무료, 시간 제한 없음
- 단점: 녹화는 유료 (Google Workspace)
- Calendly 통합: 지원
- Acuity 통합: 지원
- TutorLingua: 지원

### Microsoft Teams
- 장점: Microsoft 365 사용자에게 익숙
- 단점: 외부 사용자 초대 복잡할 수 있음
- Calendly 통합: 지원
- Acuity 통합: 지원
- TutorLingua: 지원

### Whereby
- 장점: 앱 설치 불필요, 브라우저에서 바로
- 단점: 무료 플랜 제한
- 외부 통합: 제한적

### 추천
- **대부분의 튜터**: Zoom (가장 보편적, 신뢰성 높음)
- **Google 사용자**: Google Meet
- **기업 학생 많음**: Microsoft Teams

## 문제 해결

### "Zoom 연결이 안 돼요"
1. Zoom 계정에 로그인되어 있는지 확인
2. 브라우저 캐시 지우고 다시 시도
3. 다른 브라우저로 시도
4. Zoom 앱에서 로그아웃 후 재로그인

### "링크가 이메일에 안 나와요"
1. 이벤트 유형의 Location이 "Zoom"으로 설정되어 있는지 확인
2. 이메일 템플릿에 {location} 또는 {zoom_link} 변수 포함되어 있는지 확인
3. 테스트 예약 다시 진행

### "학생이 들어오지 못해요"
1. 링크가 올바르게 복사되었는지 확인
2. 대기실 설정 확인 (학생이 대기 중일 수 있음)
3. 비밀번호가 필요한지 확인
4. Zoom 앱 업데이트 권장
