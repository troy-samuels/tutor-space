# 개인 어학 튜터가 결제를 받는 방법: 완전 가이드

"결제는 어떻게 해드릴까요?" 이 질문에 대한 답이 "음... PayPal로 보내주시면..." 이라면, 비즈니스 기회를 놓치고 있습니다.

전문적인 결제 처리는 단순히 돈을 받는 것 이상입니다:
- **신뢰 구축**: 전문적인 결제 옵션은 신뢰를 높임
- **결제 수금 개선**: 쉬운 결제 = 더 빠른 결제
- **노쇼 감소**: 선결제는 출석률을 높임
- **시간 절약**: 인보이스 추적, 리마인더 발송 불필요

이 가이드에서 개인 어학 튜터를 위한 모든 결제 옵션을 비교하고, 상황에 가장 적합한 것을 선택하는 방법을 알려드립니다.

## 결제 옵션 개요

### 옵션 1: PayPal

**가장 친숙한 국제 옵션**

PayPal은 전 세계적으로 가장 인정받는 온라인 결제 방법입니다. 학생들이 이미 계정을 가지고 있을 가능성이 높습니다.

**수수료:**
- 국내 거래: 3.49% + 고정 수수료 (₩500 또는 $0.49)
- 국제 거래: 4.49% + 고정 수수료
- 통화 변환: +2.5%

**장점:**
- 전 세계적으로 인정됨
- 학생에게 친숙함
- 구매자 보호 제공
- 인보이스 기능 포함

**단점:**
- 수수료가 높음 (특히 국제)
- 계정 동결 위험
- 환율이 불리함
- 비즈니스 계정 필요

**예시 계산 (₩50,000 수업):**
- 국내: ₩50,000 - ₩1,745 - ₩500 = ₩47,755 수령
- 국제: ₩50,000 - ₩2,245 - ₩500 = ₩47,255 수령

**적합 대상:**
- 국제 학생이 많은 튜터
- 빠른 설정이 필요한 경우
- 이미 PayPal을 사용 중인 경우

### 옵션 2: Stripe

**전문가를 위한 표준**

Stripe는 가장 인기 있는 온라인 결제 처리 서비스로, 많은 플랫폼이 Stripe를 사용합니다. TutorLingua도 Stripe를 통합합니다.

**수수료:**
- 표준: 2.9% + $0.30 (₩400)
- 국제 카드: +1.5%
- 통화 변환: +1%

**장점:**
- 낮은 수수료
- 깔끔한 체크아웃 경험
- 정기 결제 쉬움
- 자동 인보이스
- 예약 시스템과 쉽게 통합

**단점:**
- 학생이 계정 필요 없음 (카드 직접 결제)
- PayPal만큼 이름이 알려지지 않음
- 직접 설정이 기술적

**예시 계산 (₩50,000 수업):**
- 국내: ₩50,000 - ₩1,450 - ₩400 = ₩48,150 수령
- 국제: ₩50,000 - ₩2,200 - ₩400 = ₩47,400 수령

**적합 대상:**
- 전문적인 이미지를 원하는 튜터
- 예약 시스템과 통합하려는 경우
- 정기 결제/패키지를 판매하는 경우

### 옵션 3: 은행 송금

**가장 저렴하지만 가장 불편함**

직접 은행 송금은 수수료가 가장 낮지만, 국제 학생에게는 복잡합니다.

**수수료:**
- 국내: 무료 또는 최소 (₩500 이하)
- 국제: SWIFT 수수료 $15-50 + 중개은행 수수료

**장점:**
- 국내 송금 거의 무료
- 중개자 없음
- 전체 금액 수령 (국내)

**단점:**
- 국제 송금 매우 비쌈
- 송금 확인에 시간 소요 (1-5일)
- 수동 추적 필요
- 환불 복잡함

**적합 대상:**
- 국내 학생만 있는 튜터
- 수업료가 높아 수수료 비율이 낮은 경우
- 기술에 익숙하지 않은 학생

### 옵션 4: Wise (TransferWise)

**국제 결제 최적화**

Wise는 국제 송금에 특화되어 실제 환율과 낮은 수수료를 제공합니다.

**수수료:**
- 일반적으로 0.5-1% (통화에 따라 다름)
- 실제 환율 사용 (추가 마크업 없음)

**장점:**
- 매우 낮은 국제 수수료
- 실제 환율
- 여러 통화 계정 가능
- 투명한 수수료 구조

**단점:**
- 국내 송금에는 불필요
- 학생이 Wise 계정 필요
- 예약 시스템과 통합 어려움

**예시 계산 ($50 수업):**
- $50 - $0.50 (1%) = $49.50 수령
- PayPal보다 $1.50-2.00 절약

**적합 대상:**
- 국제 학생이 많은 튜터
- 유럽, 미국, 영국 학생이 있는 경우
- 환율에 민감한 경우

### 옵션 5: 통합 플랫폼 (TutorLingua)

**예약 + 결제 원스톱**

TutorLingua와 같은 올인원 플랫폼은 예약과 결제를 한 곳에서 처리합니다.

**수수료:**
- Stripe 수수료만 (2.9% + $0.30)
- 플랫폼 수수료 0% (vs. Preply 18-33%)

**장점:**
- 예약 시 자동 결제
- 학생 관리 통합
- 자동 인보이스 및 영수증
- [플랫폼 수수료 절약](/ko/blog/cluster-1-commissions/tutor-platform-susuryo-juligi)
- 패키지 및 정기 결제 쉬움

**단점:**
- 월 구독료 ($29-79)
- 플랫폼에 종속

**적합 대상:**
- 진지한 독립 튜터
- 마켓플레이스에서 전환하는 경우
- 관리 시간을 줄이고 싶은 경우

## 수수료 비교 (₩50,000 수업 기준)

| 방법 | 국내 수수료 | 국제 수수료 | 국내 수령액 | 국제 수령액 |
|------|------------|------------|-----------|-----------|
| 은행 송금 | ~₩0 | ₩20,000+ | ₩50,000 | ₩30,000 |
| Stripe | ₩1,850 | ₩2,600 | ₩48,150 | ₩47,400 |
| PayPal | ₩2,245 | ₩2,745 | ₩47,755 | ₩47,255 |
| Wise | N/A | ₩500 | N/A | ₩49,500 |

## 결제 유형별 전략

### 단일 수업 결제

**추천: Stripe 또는 PayPal**

학생이 예약할 때 바로 결제하게 하세요:
- 노쇼 감소
- 결제 추적 불필요
- 전문적인 경험

### 수업 패키지

**추천: Stripe + 예약 시스템 통합**

5회, 10회 패키지를 선결제로 판매:
- 수입 안정화
- 학생 약속 증가
- 수업당 할인 제공 가능

예시:
- 단일 수업: ₩50,000
- 5회 패키지: ₩225,000 (₩45,000/회, 10% 할인)
- 10회 패키지: ₩400,000 (₩40,000/회, 20% 할인)

### 월간 구독

**추천: Stripe 정기 결제**

정기적으로 수업을 듣는 학생에게 월간 구독 제공:
- 예측 가능한 수입
- 학생 유지율 향상
- 자동 결제 (수동 청구 불필요)

예시:
- 주 1회 (월 4회): ₩180,000/월
- 주 2회 (월 8회): ₩320,000/월

## 국제 학생 결제 팁

국제 학생이 있다면:

1. **여러 옵션 제공**: Stripe + PayPal + Wise
2. **가격을 USD/EUR로 표시**: 환율 혼란 방지
3. **환율 버퍼 추가**: 환율 변동 대비 5-10%
4. **결제 전 통화 확인**: 어떤 통화로 청구할지 명확히

## 세금 및 기록 관리

결제 방법에 상관없이:

1. **모든 거래 기록**: 스프레드시트 또는 회계 소프트웨어
2. **인보이스 발행**: 모든 결제에 대해
3. **영수증 보관**: 세금 신고용
4. **분기별 검토**: 수입 및 수수료 분석

**팁:** Stripe와 PayPal은 자동으로 거래 내역을 제공하여 기록 관리가 쉽습니다.

## 전문 인보이스 작성

좋은 인보이스에 포함해야 할 것:

- 귀하의 이름/비즈니스 이름
- 연락처
- 인보이스 번호 (순차적)
- 날짜
- 학생 이름
- 서비스 설명 (예: "60분 스페인어 수업")
- 금액 및 통화
- 결제 방법/지침
- 결제 기한

**무료 도구:**
- Wave (무료 회계 + 인보이스)
- Invoice Ninja (무료)
- Stripe (자동 인보이스)

## 결제 정책 설정

명확한 정책으로 문제를 예방하세요:

### 취소 정책
```
- 24시간 이상 전 취소: 전액 환불 또는 일정 변경
- 24시간 이내 취소: 50% 청구
- 노쇼: 전액 청구
```

### 환불 정책
```
- 미사용 패키지: 수수료 제외 환불
- 사용 중인 패키지: 사용한 수업은 정가 청구, 나머지 환불
```

### 결제 기한
```
- 단일 수업: 예약 시 선결제
- 패키지: 첫 수업 전 결제
- 월간: 매월 1일 자동 결제
```

## 결론: 단순하게 시작하고 필요에 따라 확장

완벽한 결제 시스템을 찾느라 수업 시작을 미루지 마세요:

1. **지금 당장**: PayPal 또는 은행 송금으로 시작
2. **학생 5명 이상**: Stripe 설정
3. **패키지/구독 판매**: 예약 시스템과 통합
4. **마켓플레이스에서 전환**: TutorLingua로 모든 것 통합

핵심은 **선결제를 기본으로** 하는 것입니다. 수업 후 결제를 요청하면 항상 쫓아다녀야 합니다. 예약 = 결제가 되면 삶이 훨씬 쉬워집니다.
