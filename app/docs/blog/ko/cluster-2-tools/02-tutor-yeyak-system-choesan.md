# 어학 튜터를 위한 최고의 예약 시스템: Calendly vs Acuity vs 올인원

매주 수업 스케줄을 조정하는 데 몇 시간을 보내시나요? 학생과의 끝없는 메시지 교환, 시간대 혼란, "10시 20분은 어떠세요? 아니, 11시가 더 좋을까요?" 같은 대화...

전문 예약 시스템은 이 모든 것을 바꿉니다. 당신이 자는 동안에도 학생들이 직접 예약하고, 자동 확인을 받고, 리마인더가 발송됩니다.

하지만 어떤 시스템이 어학 튜터에게 가장 적합할까요? 이 가이드에서 가장 인기 있는 옵션을 비교하여 올바른 선택을 도와드립니다.

## 왜 전용 예약 시스템이 필요한가?

수동 스케줄링의 숨겨진 비용을 계산해 보세요:

**시간 낭비:**
- 예약당 평균 15-20분의 메시지 교환
- 주 10개 예약 = 월 10-13시간 낭비
- 시급 ₩50,000이면 = 월 ₩500,000-650,000 손실

**노쇼 및 취소:**
- 리마인더 없이 노쇼율 20-30%
- 자동 리마인더로 노쇼율 5-10%로 감소

**전문성 인식:**
- "WhatsApp으로 예약"은 아마추어처럼 보임
- 전문 예약 페이지는 신뢰를 구축함

## 평가 기준

어학 튜터에게 중요한 기능을 기준으로 평가합니다:

1. **시간대 처리**: 국제 학생과 작업할 때 필수
2. **캘린더 동기화**: Google, Outlook, Apple 통합
3. **결제 통합**: 예약 시 결제 가능 여부
4. **리마인더**: 자동 이메일/SMS 알림
5. **유연성**: 다양한 수업 유형, 패키지, 반복 예약
6. **가격**: 무료 티어와 유료 플랜

## 옵션 1: Calendly

### 개요
Calendly는 가장 인기 있는 스케줄링 도구로, 간단하고 사용하기 쉽습니다.

### 장점
- **무료 티어 사용 가능**: 기본 기능 무료
- **직관적인 인터페이스**: 설정이 매우 쉬움
- **신뢰할 수 있는 시간대 변환**: 자동으로 학생 시간대 감지
- **많은 통합**: Zoom, Google Meet 자동 연결

### 단점
- **결제 통합 제한**: Professional 플랜($15/월)부터 가능
- **튜터 전용 기능 없음**: 학생 관리, 수업 이력 추적 불가
- **패키지 판매 불가**: 수업 패키지 예약 복잡

### 가격
- **무료**: 1개 이벤트 유형, 기본 기능
- **Essentials**: $10/월 - 무제한 이벤트, 사용자 정의
- **Professional**: $15/월 - 결제 통합, 리마인더 사용자 정의
- **Teams**: $20/월 - 팀 기능

### 적합 대상
- 막 시작하는 튜터
- 간단한 1:1 수업만 제공
- 별도 결제 시스템 사용

## 옵션 2: Acuity Scheduling

### 개요
Acuity는 Squarespace 소유로, Calendly보다 더 많은 기능을 제공합니다.

### 장점
- **강력한 사용자 정의**: 예약 양식 필드 사용자 정의
- **패키지 및 멤버십**: 수업 패키지 판매 가능
- **클라이언트 데이터베이스**: 기본 학생 정보 저장
- **Stripe 및 PayPal 통합**: 예약 시 결제

### 단점
- **무료 티어 없음**: 7일 평가판 후 유료
- **학습 곡선**: Calendly보다 설정이 복잡
- **인터페이스가 구식**: 디자인이 덜 현대적
- **튜터 전용 아님**: 범용 스케줄링 도구

### 가격
- **Emerging**: $16/월 - 1명 사용자, 기본 기능
- **Growing**: $27/월 - 최대 6명, 텍스트 리마인더
- **Powerhouse**: $49/월 - 최대 36명, 고급 기능

### 적합 대상
- 수업 패키지 판매가 필요한 튜터
- 더 많은 사용자 정의가 필요한 경우
- 여러 튜터가 있는 소규모 학원

## 옵션 3: TutorLingua (올인원)

### 개요
TutorLingua는 어학 튜터를 위해 특별히 구축된 플랫폼으로, 예약을 넘어 완전한 비즈니스 관리를 제공합니다.

### 장점
- **튜터 전용 설계**: 어학 수업에 최적화
- **통합 결제**: Stripe 연결로 예약 시 결제
- **학생 관리**: 수업 이력, 노트, 진행 추적
- **자동 Zoom 링크**: 예약 시 자동 생성 및 발송
- **전문 웹사이트 포함**: 별도 웹사이트 불필요
- **0% 플랫폼 수수료**: [수수료 없이 100% 유지](/ko/blog/cluster-1-commissions/tutor-platform-susuryo-juligi)

### 단점
- **범용 스케줄링 도구가 아님**: 튜터링 외 다른 용도에는 부적합
- **비교적 새로운 플랫폼**: Calendly만큼 알려지지 않음

### 가격
- **Pro**: $29/월 - 예약, 결제, 학생 관리, 웹사이트
- **Studio**: $79/월 - 위 모든 것 + 화상 수업, 녹화, AI 기능

### 적합 대상
- 진지한 독립 어학 튜터
- 플랫폼 수수료 피하고 싶은 경우
- 여러 도구를 하나로 통합하고 싶은 경우

## 시나리오별 추천

### "막 시작했고 예산이 없어요"
**추천: Calendly 무료**

간단한 스케줄링으로 시작하세요. 결제는 별도로 처리하고, 학생이 늘어나면 업그레이드를 고려하세요.

### "주 10명 이상 학생이 있고 패키지를 판매하고 싶어요"
**추천: Acuity Growing ($27/월) 또는 TutorLingua ($29/월)**

둘 다 패키지와 결제 통합을 제공합니다. 튜터 전용 기능(학생 관리, 웹사이트)이 필요하면 TutorLingua가 더 나은 가치입니다.

### "Preply나 iTalki에서 학생을 전환하고 싶어요"
**추천: TutorLingua**

[플랫폼 수수료](/ko/blog/cluster-1-commissions/preply-susuryo-seolmyeong)를 피하고 직접 관계를 구축하세요. 전문 웹사이트와 예약 시스템이 한 곳에 있어 학생에게 신뢰를 줍니다.

### "이미 웹사이트가 있고 예약만 추가하고 싶어요"
**추천: Calendly Professional ($15/월)**

기존 웹사이트에 쉽게 임베드됩니다. 간단하고 신뢰할 수 있습니다.

## 전환 팁

현재 수동 스케줄링에서 시스템으로 전환할 때:

1. **점진적으로 전환**: 모든 학생을 한 번에 바꾸지 말고, 새 학생부터 시작
2. **예약 페이지 링크 공유**: 소셜 미디어, 이메일 서명, 현재 학생에게 공유
3. **간단한 지침 제공**: "여기서 예약하세요: [링크]"
4. **시간대 명확히**: 예약 확인에 항상 시간대 포함
5. **버퍼 시간 설정**: 수업 사이 15-30분 여유 확보

## 자동 Zoom 링크 설정

어떤 시스템을 선택하든, [자동 Zoom 링크](/ko/blog/cluster-2-tools/jadong-zoom-link-haksaeng)가 필수입니다:

**Calendly:**
1. Integrations > Zoom 연결
2. 이벤트 유형 편집 > Location > Zoom 선택

**Acuity:**
1. Integrations > Zoom 연결
2. Appointment Types > 각 유형에서 Zoom 활성화

**TutorLingua:**
1. Settings > Video > Zoom 연결
2. 자동으로 모든 예약에 적용

완벽한 솔루션은 없습니다—현재 상황과 목표에 가장 적합한 것을 선택하세요:

- **시작 단계**: Calendly 무료로 시작
- **성장 단계**: Acuity 또는 TutorLingua로 업그레이드
