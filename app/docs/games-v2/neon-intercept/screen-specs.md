# Neon Intercept V2 Screen Specs

## Screen 1: Run Setup
- Elements: language selector, 4 mode tabs, immediate start state.
- Priority: one dominant action path.
- Constraint: all controls 48px minimum target size.

## Screen 2: In-Run HUD + Arena
- HUD: Time, Score, Combo, Lives.
- Arena: 3 lanes, falling clue, lane options, active feedback cue.
- Constraint: no instructional paragraph overlays during active play.

## Screen 3: Pause/Resume
- Trigger: background/visibility interruption.
- Elements: continue run, restart run.
- Constraint: preserve run timer integrity.

## Screen 4: End Sheet
- Elements: result card, share, replay, mode switching.
- Content: score, accuracy, tier, weakness direction.
- Constraint: all next actions visible without scrolling on standard mobile viewport.

## Accessibility
- Contrast AA for all text-bearing controls.
- Reduced-motion compatible with equivalent feedback meaning.
- Focus/keyboard accessibility maintained for non-touch contexts.
